Prealgebra · Lesson 6.2

Solving Linear Equations

Solve this lesson — free →All lessons

An expression like \(3x + 5\) just sits there describing a whole family of values, one for each \(x\). Set an equals sign between two expressions, like \(3x + 5 = 20\), and you make a claim that both sides come out to the same number. That claim is an equation, and the value of the variable that makes it true is a solution. Picture a two-pan balance with the equals sign as the level beam, and the variable as a gift in nested boxes you will unwrap.

A vending machine charges \(3\) tokens to wake up plus \(5\) tokens per snack, so \(w\) snacks cost \(5w + 3\). One kid spent exactly \(23\) tokens, giving \(5w + 3 = 23\). With no method yet, test whole numbers: at \(w = 3\) the left side is \(18\), at \(w = 5\) it is \(28\), so the answer sits between. The value that makes both sides equal is the solution. What whole number \(w\) makes \(5w + 3 = 23\)? Type that value.
Show solution
We want the value of \(w\) that makes \(5w+3=23\) a true statement. Testing values is a fine way in, and the pattern points straight at it. At \(w=3\) the left side is \(18\), and at \(w=5\) it is \(28\), so the value that lands on \(23\) sits between them. Test the whole number in the gap. With \(w=4\) we get \(5\times 4=20\), and \(20+3=23\). The left side equals the right side, so \(w=4\) works. Substitution check, put \(w=4\) back into the original equation. The left side is \(5\times 4+3=20+3=23\), and the right side is \(23\), so both sides match and \(w=4\) really is the solution. \(\boxed{4}\)
Before solving an equation you should be able to check one. Substituting means replacing every \(x\) with a candidate and seeing whether both sides match. A note claims \(x = 7\) solves \(3x - 5 = 19\), but \(3(7) - 5 = 16 \neq 19\), so \(7\) fails. One of the values \(7\) or \(8\) actually solves \(3x - 5 = 19\). Type the value that works.
Show solution
Checking a value means substituting it into the left side and seeing if you get \(19\). Test the note's claim first. With \(x = 7\), the left side is \(3(7) - 5 = 21 - 5 = 16\). Since \(16 \neq 19\), the candidate \(7\) fails and is not the solution. Now test the other candidate. With \(x = 8\), the left side is \(3(8) - 5 = 24 - 5 = 19\). That matches the right side exactly, \(19 = 19\), so both sides agree and \(8\) is a genuine solution. The value that solves \(3x - 5 = 19\) is \(\boxed{8}\).
Key idea
Solution and checking. An equals sign claims the two sides are the same number, and a solution is the value of the variable that makes the claim true. The identities from 6.1 held for every value, but an equation like \(2x + 5 = 11\) is choosy and usually has exactly one. You never take an answer on faith: drop the value back into both sides and watch whether the beam hangs level. If both land on the same number it is a solution; if not, it is not. Checking is the proof the solving worked.
Fun fact
The equals sign was invented by Robert Recorde in 1557. He chose two parallel lines because nothing could be more equal. The symbol took about a century to become standard.
A thermos starts at \(t\) degrees, the afternoon adds \(17\), and the display settles at \(9\), so \(t + 17 = 9\). The variable \(t\) is sealed inside an add-\(17\) box. Lift those \(17\) off the left pan, and lift the same \(17\) off the right pan to keep the scale level. Solve \(t + 17 = 9\) and type the value of \(t\).
Show solution
The variable \(t\) sits on the left, locked inside a \(+17\). To isolate it, undo that addition by subtracting \(17\) from both sides so the scale stays balanced. $$t + 17 = 9$$ $$t + 17 - 17 = 9 - 17$$ $$t = -8$$ Now check by putting \(-8\) back into the original equation. The left side becomes \(-8 + 17 = 9\), which matches the right side, so the statement is true. The value of \(t\) is \(\boxed{-8}\).
Six identical sandbags, each of unknown mass \(m\), together tip a balance to \(15\) units, so \(6m = 15\). The only thing wrapped around \(m\) is a times-\(6\), and division undoes multiplication, so divide both sides by \(6\) to leave one bag alone. The answer need not be a whole number, which is fine. Solve \(6m = 15\) and type \(m\) as a fraction in lowest terms.
Show solution
The variable \(m\) is multiplied by \(6\), so undo that by dividing both sides by \(6\). $$6m = 15$$ $$\frac{6m}{6} = \frac{15}{6}$$ $$m = \frac{15}{6}$$ The left side leaves \(m\) alone, and the right side is \(\frac{15}{6}\), which is not yet in lowest terms. Both \(15\) and \(6\) are divisible by \(3\), so \(\frac{15}{6} = \frac{15 \div 3}{6 \div 3} = \frac{5}{2}\). Check it by putting \(\frac{5}{2}\) back into the original equation. Then \(6m = 6 \cdot \frac{5}{2} = \frac{30}{2} = 15\), which matches the right side, so the balance holds. Each sandbag has mass \(\frac{5}{2}\) units. $$\boxed{5/2}$$
Key idea
The balance rule. Do the exact same operation to both sides and the beam stays level, so the value that made the equation true stays true. Adding, subtracting, multiplying, or dividing both sides by the same nonzero amount only repackages the equation, never changing the answer. The one forbidden move is touching one side and leaving the other alone, since that secretly tips the scale and tells a lie. That is the whole permission slip for solving.
In \(4x - 9 = 7\), the variable wears two boxes: it was multiplied by \(4\) first, then had \(9\) taken away. To unwrap, peel in reverse and pull the outer box off first, so add \(9\) to both sides, then divide by \(4\). Solve \(4x - 9 = 7\) and type the value of \(x\).
Show solution
Peel the outer box first by adding \(9\) to both sides. $$4x - 9 + 9 = 7 + 9$$ The left side becomes \(4x\) and the right side becomes \(16\), so \(4x = 16\). Now the variable wears only the times-\(4\) box, so divide both sides by \(4\). $$\frac{4x}{4} = \frac{16}{4}$$ This gives \(x = 4\). Check by substituting back into the original equation. \(4(4) - 9 = 16 - 9 = 7\), which matches the right side, so it holds. The value of \(x\) is \(\boxed{4}\).
Start with \(n\), multiply by \(-3\), then add \(2\), and you reach \(10\), so \(-3n + 2 = 10\). The times-\(-3\) is the inner box and the add-\(2\) is the outer box, so peel from the outside in: subtract \(2\) first, then divide by \(-3\). Watch the signs, because the answer will not be a whole number. Solve \(-3n + 2 = 10\) and type \(n\) as a fraction in lowest terms, including its sign.
Show solution
Peel the outer box first. Subtract \(2\) from both sides. $$-3n + 2 - 2 = 10 - 2$$ The left side collapses to \(-3n\) and the right side becomes \(8\), so \(-3n = 8\). Now undo the inner box by dividing both sides by \(-3\). $$\frac{-3n}{-3} = \frac{8}{-3}$$ On the left the \(-3\) cancels and leaves \(n\). On the right, a positive over a negative is negative, and \(8\) and \(3\) have no common factor, so it does not reduce. That gives \(n = -\tfrac{8}{3}\). A negative fraction is a perfectly good solution, the equation does not mind. Check it by substituting back. With \(n = -\tfrac{8}{3}\), the term \(-3n\) is \(-3 \cdot \left(-\tfrac{8}{3}\right) = 8\), and \(8 + 2 = 10\), which matches the right side. The solution is \(\boxed{-8/3}\).
Key idea
Peel from the outside in. To free a variable, undo the operations stacked around it in reverse order, the last thing done first, like taking off your coat before your shirt. A constant added on the outside comes off before the number multiplying on the inside, and each peel is the opposite operation done to both sides at once, so the equation stays balanced and the answer never moves. When the variable stands bare, plug your number back in to be sure both sides agree.

Real equations are not always neat. Sometimes the variable's side is piled with several terms, like \(2x + 3x\), or hides a group waiting to be handed out, like \(4(t - 7)\). Before peeling a single layer, tidy that side first with the housekeeping from 6.1, gathering like terms and distributing groups. In every equation here the variable stays on one side and the other side stays a plain constant.

A delivery rider's day reads \(5k + 2 + 3k - 7 = 19\), where \(k\) is the number of trips. Do not undo anything yet. First tidy the cluttered left side by combining like terms, exactly as in 6.1, then isolate \(k\). Solve \(5k + 2 + 3k - 7 = 19\) and type the value of \(k\).
Show solution
Tidy the left pan first by combining like terms, exactly as in 6.1. The \(k\) terms gather as \(5k + 3k = 8k\), and the plain numbers gather as \(2 - 7 = -5\), so the whole left side becomes \(8k - 5\). Notice the variable lived only on the left the whole time, so now the equation is a clean two-step. $$8k - 5 = 19$$ Add \(5\) to both sides to peel off the outer number. $$8k = 24$$ Divide both sides by \(8\). $$k = 3$$ Check by substituting \(k = 3\) back into the original left side, \(5(3) + 2 + 3(3) - 7 = 15 + 2 + 9 - 7 = 19\), which matches the right side, so the scale truly balances. The rider made \(\boxed{3}\) trips.
A side can hide its real shape inside a group. In \(6(2 - q) = 30\), the outer times-\(6\) has to reach both pieces inside before anything else can happen, so it meets the \(2\) and the \(-q\), and that minus sign rides along. Distribute carefully, then peel the two-step equation. Solve \(6(2 - q) = 30\) and type the value of \(q\).
Show solution
Start by distributing the outer \(6\) across the group, giving it to both the \(2\) and the \(-q\). $$6(2 - q) = 30$$ $$12 - 6q = 30$$ Notice the \(-q\) became \(-6q\), the minus sign came along for the ride. Now it is a clean two-step. Subtract \(12\) from both sides to strip the constant off the variable side. $$-6q = 18$$ Then divide both sides by \(-6\). A positive over a negative is negative. $$q = \frac{18}{-6} = -3$$ Check it by putting \(-3\) back into the original. Inside the group, \(2 - (-3) = 5\), and \(6 \times 5 = 30\), which matches the right side, so the equation holds. The value of \(q\) is \(\boxed{-3}\).
Key idea
Simplify before peeling. Before touching both sides, clean up the side holding the variable: distribute any group with a number out front and gather every like term, until that side is as short as it goes, ideally one variable term and one plain number. A pan piled with loose terms or a sealed group is almost always a plain two-step once tidied. So the real first step of solving is the simplifying from 6.1 doing its job again, and only then do you start peeling.
A paint mixer uses two thirds of a tank to fill today's order, so \(\frac{2}{3}x = 10\), where \(x\) is the tank's capacity in liters. The variable is wrapped in a multiply-by-\(\frac{2}{3}\) box, so multiply both sides by the reciprocal \(\frac{3}{2}\), which turns the \(\frac{2}{3}\) into a plain \(1\). Solve \(\frac{2}{3}x = 10\) and type the value of \(x\).
Show solution
The variable is multiplied by \(\frac{2}{3}\), so multiply both sides by the reciprocal \(\frac{3}{2}\) to undo it. $$\frac{3}{2} \cdot \frac{2}{3}x = \frac{3}{2} \cdot 10$$ On the left, \(\frac{3}{2} \cdot \frac{2}{3} = 1\), so the left side is just \(x\). On the right, \(10 \cdot \frac{3}{2} = \frac{30}{2} = 15\). $$x = 15$$ If you prefer, the same arithmetic comes from multiplying both sides by \(3\) to get \(2x = 30\), then dividing by \(2\) to get \(x = 15\). Either route lands in the same place. Check it by substituting \(x = 15\) back in. The left side is \(\frac{2}{3} \cdot 15 = \frac{30}{3} = 10\), which matches the right side, so the tank holds \(\boxed{15}\) liters.
Key idea
Clear denominators first. When the variable is trapped under a fraction bar, the denominator is the outermost box, so make it disappear by multiplying both sides by it; the division cancels and you are left with whole-number pieces. This works whether the denominator wraps a whole side, like \(\frac{x+5}{4} = 3\), or appears as a fractional coefficient, like \(\frac{2}{3}x = 8\), where multiplying by the flipped fraction does the same job. It is still a fair both-sides move, so the answer never changes.
Solving for the variable is often just the first half of a question. A ticket booth's group passes \(x\) satisfy \(5(x - 2) = 35\), but the manager wants the wristband count, given by \(2x + 1\). First solve \(5(x - 2) = 35\) for \(x\), then compute \(2x + 1\). Type that final number.
Show solution
First solve the equation for \(x\). The left side has the whole group \(x-2\) multiplied by \(5\), so divide both sides by \(5\) to peel that off. $$5(x-2)=35$$ $$x-2=7$$ Now add \(2\) to both sides to get \(x\) by itself. $$x=9$$ Check this in the original equation by substituting \(x=9\). The left side becomes \(5(9-2)=5\times 7=35\), which matches the right side, so \(x=9\) is correct. But the question does not want \(x\). It wants the number of wristbands, which the rule writes as \(2x+1\). Substitute \(x=9\) into that expression. $$2x+1=2\times 9+1=18+1=19$$ So the booth hands out \(19\) wristbands. The final value is \(\boxed{19}\).
Sometimes you are handed the variable and must hunt for a missing number. A game designer's win happens when a score \(x\) satisfies \(3x + c = 20\), where \(c\) is a fixed bonus she still has to choose. She wants \(x = 4\) to count as a win, which means \(x = 4\) must be a solution. Substitute \(4\) for \(x\), then solve for \(c\). Type the value of \(c\).
Show solution
Saying \(x = 4\) is a solution means \(x = 4\) must make \(3x + c = 20\) a true statement. So substitute \(4\) in for \(x\). $$3(4) + c = 20$$ Multiply first to clean up the left side. $$12 + c = 20$$ Now \(c\) is alone on one side, and you isolate it by subtracting \(12\) from both sides. $$12 + c - 12 = 20 - 12$$ $$c = 8$$ Check it by putting both \(x = 4\) and \(c = 8\) back into the original rule. The left side is \(3(4) + 8 = 12 + 8 = 20\), and the right side is \(20\), so both sides land on \(20\) and the win condition holds. The fixed bonus the designer needs is \(\boxed{8}\).
Key idea
Always check. Checking is the second half of solving, not a chore tacked on. Put the value you found back into the original equation, the messy one before you cleared or combined anything, and work out each side on its own. If both sides land on the same number, the equation is balanced and you know you are right. If they do not match, you have caught yourself, almost always a dropped negative or a small slip, before anyone else saw it.
A community garden shares a total across \(4\) equal plots, ending with \(6\) beds each, so \(\dfrac{3(m-2)+2m}{4} = 6\), where \(m\) is beds per row. Peel the outermost box first by multiplying both sides by \(4\), then distribute, combine like terms, and isolate \(m\). Solve \(\dfrac{3(m-2)+2m}{4} = 6\) and type the value of \(m\).
Show solution
Start with the equation and peel off the outermost box first. The entire left side is divided by 4, so undo that by multiplying both sides by 4. $$\frac{3(m-2)+2m}{4}=6$$ $$3(m-2)+2m=24$$ Now work inside. Distribute the 3 across \(m-2\) to get \(3m-6\), then bring down the \(2m\). $$3m-6+2m=24$$ Combine the like terms \(3m\) and \(2m\) into \(5m\), so the left side collapses. $$5m-6=24$$ Keep peeling from the outside. The \(-6\) is the last thing added on, so add 6 to both sides. $$5m=30$$ Finally \(m\) is multiplied by 5, so divide both sides by 5. $$m=6$$ The variable rode along on one side the whole way. Check by substituting \(m=6\) back into the original. The top becomes \(3(6-2)+2(6)=3\cdot 4+12=12+12=24\), and \(24\div 4=6\), which matches the right side. $$\boxed{6}$$

Review problems

Review 1. A diver's instrument reads higher as she rises, and her start reading \(x\) satisfies \(x + 12 = 5\), where negative values mean below the surface. Solve \(x + 12 = 5\) and type the value of \(x\).
Show solution
The variable \(x\) sits alone on the left, with \(12\) added to it, so I undo that addition. Subtract \(12\) from both sides to keep the equation balanced. $$x + 12 = 5$$ $$x + 12 - 12 = 5 - 12$$ $$x = -7$$ So \(x = \boxed{-7}\). Checking it, substitute \(-7\) back into the left side. That gives \(-7 + 12 = 5\), which matches the right side, so the reading is correct.
Review 2. A jar holds some marbles, and after \(6\) more are dropped in it holds \(20\). With \(n\) the starting count, that is \(n + 6 = 20\). Solve for \(n\) and type the value.
Show solution
The variable \(n\) sits on the left with a \(6\) added to it, so to get \(n\) by itself, undo that addition by subtracting \(6\) from both sides. $$n + 6 = 20$$ $$n + 6 - 6 = 20 - 6$$ $$n = 14$$ So the jar started with \(\boxed{14}\) marbles. Checking it, \(14 + 6 = 20\), which matches the right side, so the value works.
Review 3. Four identical packs together hold \(9\) mini sets, so \(4m = 9\), where \(m\) is how many fit in one pack. Solve for \(m\) and type the value as a fraction in lowest terms.
Show solution
We start with \(4m = 9\). Since \(m\) is multiplied by 4, we undo that by dividing both sides by 4. That gives \(m = \tfrac{9}{4}\). To check, put it back in and see if \(4 \cdot \tfrac{9}{4} = 9\), and it does. Since 9 and 4 have no common factor other than 1, the fraction is already in lowest terms, so \(\boxed{9/4}\).
Review 4. A descending drone's tracker satisfies \(-6y = 54\). Divide both sides by \(-6\) to solve. Type the value of \(y\), including its sign.
Show solution
We start with \(-6y = 54\). The \(y\) is multiplied by \(-6\), so we undo that by dividing both sides by \(-6\). On the left that leaves just \(y\). On the right we compute \(54\) divided by \(-6\). A positive divided by a negative is negative, and \(54\) divided by \(6\) is \(9\), so the result is \(-9\). That gives \(y = -9\). Check it by putting it back in. \(-6\) times \(-9\) is \(54\), which matches the right side, so the answer is \(\boxed{-9}\).
Review 5. A vending machine charges \(x\) per snack plus a flat \(\$7\) fee, and \(3\) snacks come to \(\$22\), so \(3x + 7 = 22\). Peel the constant, then the coefficient. Solve and type the value of \(x\).
Show solution
Start with the equation and undo the operations in reverse, peeling off the constant first and then the coefficient. Subtract \(7\) from both sides to clear the added constant. $$3x + 7 - 7 = 22 - 7$$ $$3x = 15$$ Now \(3x\) means \(3\) times \(x\), so divide both sides by \(3\). $$\frac{3x}{3} = \frac{15}{3}$$ $$x = 5$$ Check by substituting \(x = 5\) back into the original equation. The left side becomes \(3(5) + 7 = 15 + 7 = 22\), which matches the right side, so the value works. \(\boxed{5}\)
Review 6. A baker uses \(w\) cups on each of two trays and sets aside \(3\), using \(4\) cups in all, so \(2w - 3 = 4\). Add \(3\) to both sides, then divide by \(2\). Type \(w\) as a fraction in lowest terms.
Show solution
Start with \(2w - 3 = 4\). The variable term has a number hanging off it, so peel that away first. Add 3 to both sides and the left becomes just \(2w\), while the right becomes \(4 + 3 = 7\). That leaves \(2w = 7\). Now \(w\) is only being multiplied by 2, so divide both sides by 2 to get \(w = \frac{7}{2}\). Check it by putting \(\frac{7}{2}\) back in. Then \(2 \cdot \frac{7}{2} = 7\), and \(7 - 3 = 4\), which matches the right side, so it holds. The dough on each tray is \(\boxed{7/2}\) cups.
Review 7. Tidy the busy left side first, then hunt for the value. Solve \(7t - 2 - 4t + 5 = 18\) and type the value of \(t\).
Show solution
Start by cleaning up the left side, since the variable only lives over there. Group the \(t\) terms and the constants separately. $$7t - 4t = 3t \quad\text{and}\quad -2 + 5 = 3$$ So the equation simplifies to a much friendlier shape. $$3t + 3 = 18$$ Now isolate \(t\). Subtract \(3\) from both sides to clear the constant. $$3t = 15$$ Then divide both sides by \(3\). $$t = 5$$ Check it by putting \(5\) back into the original left side. We get \(7\cdot 5 - 2 - 4\cdot 5 + 5 = 35 - 2 - 20 + 5 = 18\), which matches the right side. So the solution is \(\boxed{5}\).
Review 8. Five identical kits each ship with \(4\) items missing, so each holds \(d - 4\), and all five total \(15\), giving \(5(d - 4) = 15\). Distribute the \(5\) first, then isolate \(d\). Type the value of \(d\).
Show solution
Start by distributing the 5 across the parentheses so the left side is a clean sum. $$5(d-4)=15$$ $$5d-20=15$$ Now peel away the pieces around \(d\). Add 20 to both sides to clear the \(-20\). $$5d=35$$ Divide both sides by 5 to free \(d\). $$d=7$$ So \(\boxed{7}\). Check it by going back to the original equation. With \(d=7\) each kit holds \(7-4=3\) items, and five kits hold \(5\cdot 3 = 15\), which matches the 15 items counted, so the value works.
Review 9. A triangle has three equal edges, each measuring \(2k + 5\), and its perimeter is \(9\), so \(3(2k + 5) = 9\). Distribute, then solve. Type the value of \(k\).
Show solution
Start by distributing the \(3\) across the group. \(3(2k+5)\) becomes \(6k+15\), so the equation is now \(6k+15=9\). Undo the \(+15\) by subtracting \(15\) from both sides, which leaves \(6k=-6\). Then divide both sides by \(6\) to get \(k=-1\). $$3(2k+5)=9$$ $$6k+15=9$$ $$6k=-6$$ $$k=-1$$ Check it by putting \(-1\) back in. \(2(-1)+5=3\), and \(3 \cdot 3=9\), which matches the right side. So \(\boxed{-1}\).
Review 10. Three fifths of a tank's capacity \(x\) comes to \(9\) liters, so \(\frac{3}{5}x = 9\). Multiply both sides by the reciprocal \(\frac{5}{3}\) to free \(x\). Type the value of \(x\).
Show solution
The variable \(x\) is multiplied by \(\frac{3}{5}\), so I undo that by multiplying both sides by the reciprocal \(\frac{5}{3}\). $$\frac{5}{3} \cdot \frac{3}{5}x = \frac{5}{3} \cdot 9$$ On the left the two fractions cancel to \(1\), leaving \(x\). On the right \(\frac{5}{3} \cdot 9 = \frac{45}{3} = 15\). $$x = 15$$ Check by putting \(15\) back in. \(\frac{3}{5} \cdot 15 = \frac{45}{5} = 9\), which matches the right side, so it holds. \(\boxed{15}\)
Review 11. A fractional coefficient scales \(b\), and the other side is negative: \(\tfrac{2}{7}b = -4\). Free \(b\) by multiplying both sides by the flipped fraction \(\tfrac{7}{2}\), carrying the negative through. Type the value of \(b\).
Show solution
The variable \(b\) is being multiplied by \(\tfrac{2}{7}\), so to isolate it I multiply both sides by the reciprocal \(\tfrac{7}{2}\). On the left those two fractions cancel to \(1\), and on the right I carry out the multiplication, keeping the sign. $$\tfrac{7}{2}\cdot\tfrac{2}{7}b = \tfrac{7}{2}\cdot(-4)$$ $$b = \tfrac{7\cdot(-4)}{2} = \tfrac{-28}{2} = -14$$ So \(b = -14\). Check it by substituting back into the original left side. \(\tfrac{2}{7}\cdot(-14) = \tfrac{2\cdot(-14)}{7} = \tfrac{-28}{7} = -4\), which matches the right side, so the value works. The solution is \(\boxed{-14}\).
Review 12. A value \(3h + 2\) shared equally among \(4\) display cases gives \(5\) each, so \(\dfrac{3h+2}{4} = 5\). Undo the division first by multiplying both sides by \(4\), then solve. Type the value of \(h\).
Show solution
The variable sits inside a fraction, and the whole top is divided by \(4\). Undo that division first by multiplying both sides by \(4\). $$\frac{3h+2}{4}=5$$ $$3h+2=20$$ Now the variable has two layers wrapped around it. A \(2\) is added, and the result is multiplied by \(3\). Peel them off in reverse order. Subtract \(2\) from both sides. $$3h=18$$ Then divide both sides by \(3\). $$h=6$$ Check by putting \(6\) back in. The top becomes \(3\cdot 6+2=20\), and \(\frac{20}{4}=5\), which matches the right side. So \(\boxed{6}\).
Review 13. A quantity \(5x + 2\) divided into \(4\) equal parts gives \(6\) each, so \(\frac{5x+2}{4} = 6\). Clear the denominator first by multiplying both sides by \(4\), then solve. Type \(x\) as a fraction in lowest terms.
Show solution
The variable sits over a denominator, so clear that first. Multiply both sides by \(4\). $$\frac{5x+2}{4}\cdot 4 = 6\cdot 4$$ $$5x+2 = 24$$ Now subtract \(2\) from both sides. $$5x = 22$$ Divide both sides by \(5\). $$x = \frac{22}{5}$$ Since \(22\) and \(5\) share no common factor, this is already in lowest terms. Check it by putting \(\frac{22}{5}\) back in. Then \(5x = 5\cdot\frac{22}{5} = 22\), so \(5x+2 = 24\), and \(\frac{24}{4}=6\), which matches the right side. The answer is \(\boxed{22/5}\).
Review 14. You are told \(n = 5\) is a solution of \(2n + c = 17\), so plugging in \(5\) must make it true. Substitute \(n = 5\), then solve for the constant \(c\). Type the value of \(c\).
Show solution
A solution is a value of the variable that makes the equation true, so saying \(n = 5\) is a solution means \(2n + c = 17\) has to hold when we put 5 in for \(n\). Substitute first. $$2(5) + c = 17$$ $$10 + c = 17$$ Now isolate \(c\) by subtracting 10 from both sides. $$10 + c - 10 = 17 - 10$$ $$c = 7$$ Check it by going back to the original equation with \(n = 5\) and \(c = 7\). The left side is \(2(5) + 7 = 10 + 7 = 17\), which matches the right side, so the constant is \(\boxed{7}\).
