Prealgebra · Lesson 8.1

Squares and Their Roots

Solve this lesson — free →All lessons

Squaring a side gives the area, so \(5^2 = 25\). This lesson flips the question. Given that a panel uses exactly 36 tiles in a square, how long was one edge? That backward move, recovering the side from the area, is the square root.

A square concrete pad covers \(196\) square feet. Find the length of one edge in feet.
Show solution
The area of a square is the side length times itself, so finding the side means undoing that square. We want the whole number whose square is \(196\), which is exactly \(\sqrt{196}\). The landmarks pin it down. We know \(13^2 = 169\), which falls short of \(196\), and \(15^2 = 225\), which overshoots it, so the side must be the one whole number squeezed between \(13\) and \(15\). That number is \(14\). Checking it, \(14 \times 14 = 196\), which matches the area exactly. So \(\sqrt{196} = 14\), and one edge of the pad is \(\boxed{14}\) feet long.
Key idea
Square root. The square root of a number \(n\), written \(\sqrt{n}\) with the radical symbol drawn as a hook and a bar over the \(n\), is the number whose square is \(n\). Squaring multiplies a number by itself, and the square root reverses that, carrying you from the result back to the number you started with. So \(\sqrt{196} = 14\) precisely because \(14^2 = 196\). When the area of a square is a perfect square, its square root is exactly the length of one side.
Key idea
Perfect square. A perfect square is a whole number you get by multiplying some whole number by itself, the way \(81 = 9^2\) and \(144 = 12^2\). Taking the square root of a perfect square just asks which number was multiplied by itself to make it, and that number always comes out a clean whole number. Once you know the squares of the small whole numbers, roughly \(1\) through \(20\), reading off a root is mostly recognizing a square you have already met and naming the number hiding underneath.
A video wall is built from \(625\) identical square panels in a perfect square. Since \(20^2 = 400\) and \(30^2 = 900\), the side length is in the twenties. What is \(\sqrt{625}\)?
Show solution
The square root of \(625\) is the nonnegative number whose square is \(625\). The landmark already narrows it down, because \(20^2 = 400\) is too small and \(30^2 = 900\) is too big, so the answer sits in the twenties. Trying the middle of that range, \(25 \times 25 = 625\) exactly. So one full side of the video wall is \(25\) panels long, and \(\sqrt{625} = 25\). \(\boxed{25}\)
Both \(10\) and \(-10\) square to \(100\), so two numbers satisfy \(x^2 = 100\). But the radical \(\sqrt{100}\) names only one of them, the nonnegative one. Evaluate \(\sqrt{100}\).
Show solution
Two numbers square to 100, namely 10 and \(-10\), since \(10^2 = 100\) and \((-10)^2 = 100\). The radical symbol is defined to name only the nonnegative one, the value that is zero or positive. Between 10 and \(-10\), the nonnegative choice is 10. So \(\sqrt{100} = 10\), never \(-10\). The symbol points to exactly one value, and that value is \(\sqrt{100} = \boxed{10}\).
Key idea
Principal square root. A number and its negative square to the same positive result, since \(10^2 = 100\) and \((-10)^2 = 100\), so \(\sqrt{100}\) would point at two values if we let it mean either one. By convention the radical names only the nonnegative value, called the principal square root, so \(\sqrt{100} = 10\) and never \(-10\). That single rule turns the square root into a clean machine: feed it a perfect square and it hands back exactly one answer every time, and that answer is never negative.

The radical \(\sqrt{36}\) names a single value, \(6\), the nonnegative side of a square with area \(36\), and there is nothing ambiguous about where it points. But a second question wears almost the same clothes. Instead of naming a value, \(x^2 = 36\) sets up an equation and asks for every number whose square lands on \(36\). Same number, two questions, two different shapes of answer. The trick from here is reading carefully enough to tell which one you have been handed.

The equation \(x^2 = 49\) asks for every number whose square is \(49\). Since \(7^2 = 49\) and \((-7)^2 = 49\), both \(7\) and \(-7\) work. How many values of \(x\) satisfy \(x^2 = 49\)?
Show solution
We want every value of \(x\) with \(x^2 = 49\). Testing the obvious one, \(7^2 = 49\), so \(x = 7\) works. Now think about negatives. Since a negative times a negative is positive, \((-7)^2 = (-7)\cdot(-7) = 49\), so \(x = -7\) works too. No other whole number squares to \(49\), because anything bigger than \(7\) overshoots and anything between gives a different result. That leaves exactly two values, \(7\) and \(-7\). Notice this is more than \(\sqrt{49}\) gives on its own, since the radical reports only the nonnegative \(7\). The equation, by contrast, accepts both. The count of solutions is \(\boxed{2}\).
Key idea
Equation vs. radical. When \(n\) is positive, asking which numbers satisfy \(x^2 = n\) has two answers, a positive one and its negative twin, because squaring throws away the sign and both land in the same place. But \(\sqrt{n}\) names only the nonnegative one of the two. So solving \(x^2 = 36\) gives \(6\) and \(-6\), while evaluating \(\sqrt{36}\) gives just \(6\). The equation can hand you two values; the radical always names exactly one. Extra context, like knowing \(x\) is negative, picks which of the two you want.
A depth sensor satisfies \(x^2 = 324\). Two values work: \(x = 18\) and \(x = -18\). Since depth is negative, which value is \(x\)?
Show solution
Start with the equation \(x^2 = 324\). To find how big \(x\) is, take the square root of \(324\). Since \(18^2 = 324\), we get \(\sqrt{324} = 18\), so \(x\) has size \(18\). Squaring hides the sign, because both \(18^2 = 324\) and \((-18)^2 = 324\), so the equation alone leaves two candidates, \(x = 18\) and \(x = -18\). Now use the story to choose. The value \(x\) records a depth below the surface, which means \(x\) must be negative, so \(18\) cannot be the answer. That leaves the negative candidate, and a quick check confirms \((-18)^2 = 324\). So \(x = \boxed{-18}\).
Squaring any real number gives a result that is zero or positive. So how many real values of \(x\) satisfy \(x^2 = -9\)?
Show solution
Square any real number and check the sign. A positive number squared is positive, a negative number squared is positive, and \(0^2 = 0\). That covers every real number, so a square is always \(0\) or greater. The equation \(x^2 = -9\) wants a square that equals \(-9\), which sits below zero. Nothing real can reach down there, so no real value of \(x\) satisfies it. The count of real solutions is \(\boxed{0}\).
Key idea
No real root of a negative. Squaring any real number, positive or negative or zero, gives a result that is zero or positive, never negative, since a negative times a negative flips back to positive and a positive times a positive stays positive. So nothing you put in for \(x\) makes \(x^2\) negative, which means a negative number has no real square root and an equation like \(x^2 = -25\) has no real solution. This is not a gap in your tools, it is just how squaring works: every square points the same way, away from the negatives.
Evaluate \(\sqrt{(-12)^2}\). Square first, then take the root. What whole number do you get?
Show solution
Work from the inside out. The square happens first, so compute \((-12)^2\). A negative times a negative is positive, so \((-12)^2 = 144\), a positive number. Now the radical takes over, and it always returns the nonnegative root, so \(\sqrt{144} = 12\) since \(12^2 = 144\). Notice what happened. We started with \(-12\), but squaring threw the sign away, and the radical handed back the nonnegative version. That is why \(\sqrt{(-12)^2}\) comes out positive even though the number inside began negative. The value is \(\boxed{12}\).
Key idea
Cancellation rule. Squaring and taking a square root are opposite moves, so doing one then the other lands you back where you started: \(\left(\sqrt{n}\right)^2 = n\) and \(\sqrt{n^2} = n\) whenever \(n\) is nonnegative. The one thing to watch is that the radical only hands back a nonnegative value, so when you square a negative number first, the square erases its sign and the root can only return the positive twin. That is why \(\sqrt{(-12)^2} = 12\) and not \(-12\): the squaring wiped the sign away and the root had no way to put it back.
A square garden covers \(1{,}296\) sq ft. Notice \(1{,}296 = 16 \times 81\). Use \(\sqrt{1{,}296} = \sqrt{16} \times \sqrt{81}\) to find the side length.
Show solution
Break the big number into perfect-square factors. Since \(1{,}296 = 16 \times 81\), the root splits into a product of roots, so \(\sqrt{1{,}296} = \sqrt{16} \times \sqrt{81}\). Each piece is easy, with \(\sqrt{16} = 4\) and \(\sqrt{81} = 9\). Multiplying gives \(4 \times 9 = 36\). To be sure, check that \(36^2 = 36 \times 36 = 1{,}296\), which matches the area, so the side length is exactly \(\sqrt{1{,}296} = \boxed{36}\) feet.
Key idea
Product rule for roots. \(\sqrt{ab} = \sqrt{a} \times \sqrt{b}\) for nonnegative \(a\) and \(b\). When a perfect square is too big to recognize on sight, split it into a product of two smaller perfect squares you already know, root each piece, then multiply, so \(\sqrt{1{,}296} = \sqrt{16} \times \sqrt{81} = 4 \times 9 = 36\). This works because rooting and multiplying cooperate, but only when both pieces really are perfect squares, since that is what keeps every root a clean whole number.
A classmate claims \(\sqrt{1024} = 34\). But \(34^2 = 1156 > 1024\), and a number ending in 4 squares to something ending in 6, not 4. What is the correct value of \(\sqrt{1024}\)?
Show solution
Both tests already narrow things down before any real arithmetic. The size test puts the root below 34, since \(34^2 = 1{,}156\) overshoots \(1{,}024\). The last-digit test rules out any root ending in 4, since those square to a number ending in 6, while \(1{,}024\) ends in 4. A root ending in 2 fits the last digit, because \(2 \times 2 = 4\). The candidate just under 34 that ends in 2 is 32. Squaring it gives \(32^2 = 32 \times 32 = 1{,}024\), an exact hit, so \(1{,}024\) is a perfect square and its root is the nonnegative number that squares to it. Therefore \(\sqrt{1024} = \boxed{32}\).
Fun fact
The word radical comes from Latin radix (root), the same as the vegetable radish. Rudolff used \(\sqrt{\phantom{n}}\) in 1525. The bar on top, the vinculum, marks what sits under the root.
A square herb bed has a side of \(15\) feet, so \(\sqrt{x} = 15\) where \(x\) is the area. Square both sides to free \(x\). What is \(x\)?
Show solution
The equation is \(\sqrt{x} = 15\), and \(x\) is hiding under the radical. To free it, undo the square root by squaring, and to keep the equation balanced you square both sides. $$\left(\sqrt{x}\right)^2 = 15^2$$ Squaring a square root cancels it, so the left side becomes just \(x\). On the right, \(15^2 = 15 \times 15 = 225\). That gives \(x = 225\). To be sure, plug it back in. The square root of \(225\) really is \(15\), since \(15 \times 15 = 225\), so the side length checks out and the area is \(225\) square feet. \(\boxed{225}\)
Key idea
Solving \(\sqrt{x} = k\). When the unknown is trapped under a radical, free it by squaring both sides, since squaring is the exact move that undoes a square root: \((\sqrt{x})^2 = k^2\), giving \(x = k^2\). This only works when \(k \ge 0\), because the radical names a value that is zero or positive and can never equal a negative number. If an equation sets \(\sqrt{x}\) equal to something negative, it has no solution, even when the algebra looks like it should go through.
Solve \(\sqrt{3x + 4} = 11\). Square both sides, then solve the resulting linear equation. What is \(x\)?
Show solution
Start with \(\sqrt{3x + 4} = 11\). The right side is positive, so a real solution can exist. The square root and the square undo each other, so squaring both sides frees the inside. That gives \(3x + 4 = 11^2 = 121\). Now it is a plain linear equation. Subtract \(4\) from both sides to get \(3x = 117\), then divide by \(3\) to get \(x = 39\). Check it by substituting back. Inside the radical you get \(3 \times 39 + 4 = 117 + 4 = 121\), and \(\sqrt{121} = 11\), which matches. So the value of \(x\) is \(\boxed{39}\).
How many solutions does \(\sqrt{x - 2} = -4\) have? The radical is always nonnegative, can it ever equal \(-4\)?
Show solution
The radical \(\sqrt{\phantom{x}}\) stands for the principal square root, which is always zero or positive. The right side of the equation is \(-4\), a negative number, and a principal square root can never equal a negative number. So before doing any algebra you can already suspect that nothing works. Let us confirm it. Squaring both sides gives \(x - 2 = 16\), so \(x = 18\). Now check that candidate in the original equation. \(\sqrt{18 - 2} = \sqrt{16} = 4\), and \(4\) is not \(-4\). The value squaring produced does not satisfy the equation, because squaring can erase a sign and sneak in a false answer. With that lone candidate rejected, no value of \(x\) is left that makes the equation true. The number of solutions is \(\boxed{0}\).

Review problems

Review 1. A square garden bed covers \(121\) sq ft. What is the side length, in feet? Evaluate \(\sqrt{121}\).
Show solution
The square root of a number is the nonnegative value that squares to give it. We are looking for the number whose square is \(121\). Testing nearby whole numbers, \(10^2 = 100\) is too small and \(12^2 = 144\) is too big, so the root is between \(10\) and \(12\). Checking \(11\), we get \(11 \times 11 = 121\), which matches exactly. So \(\sqrt{121} = 11\), and that is the side length of the garden bed in feet. The answer is \(\boxed{11}\).
Review 2. A square rug covers \(256\) sq ft. What is its side length in feet?
Show solution
The area of a square equals its side length times itself, so the side length is the square root of the area. That means I need \(\sqrt{256}\), the nonnegative number whose square is \(256\). Testing nearby whole numbers, \(15^2 = 225\) is too small and \(17^2 = 289\) is too large, so the root must be \(16\). Checking, \(16 \times 16 = 256\), which matches the area exactly. The side length of the rug is \(\boxed{16}\) feet.
Review 3. A square patio covers \(169\) sq ft. Since \(10^2 = 100\) and \(20^2 = 400\), the side length is between \(10\) and \(20\). Find \(\sqrt{169}\).
Show solution
Taking a square root asks for the nonnegative number whose square is \(169\), the same as recovering the side of a square from its area. The landmarks pin it down fast. Since \(10^2 = 100\) and \(20^2 = 400\), the root is between \(10\) and \(20\), so I only need to search that gap. Because \(169\) ends in a \(9\), the root must end in \(3\) or \(7\). Checking the smaller option, \(13 \times 13 = 169\), which matches exactly. So \(\sqrt{169} = 13\), meaning each side of the patio is \(13\) feet long. \(\boxed{13}\)
Review 4. When you write \(x^2 = 9\), both \(3\) and \(-3\) work. But \(\sqrt{9}\) names only the nonnegative one. Evaluate \(\sqrt{9}\).
Show solution
The equation \(x^2 = 9\) has two solutions, \(3\) and \(-3\), because each of them squares to \(9\). The radical \(\sqrt{9}\) is different from that equation. By definition it names only the nonnegative number whose square is \(9\), so it picks out exactly one value and ignores the negative one. Since \(3 \times 3 = 9\) and \(3\) is nonnegative, we get \(\sqrt{9} = 3\). So the value is \(\boxed{3}\).
Review 5. A square garden covers \(400\) sq ft. What is \(\sqrt{400}\)?
Show solution
I want the nonnegative number whose square is \(400\). One clean way is to split \(400\) into perfect-square factors, since \(400 = 4 \times 100\). The root of a product of perfect squares is the product of their roots, so \(\sqrt{400} = \sqrt{4} \times \sqrt{100} = 2 \times 10 = 20\). I can check this by squaring back, and \(20^2 = 20 \times 20 = 400\), which matches the area. So the side of the garden is \(\sqrt{400} = \boxed{20}\) feet.
Review 6. Both \(9\) and \(-9\) square to \(81\), since \(9^2 = 81\) and \((-9)^2 = 81\). How many values of \(x\) satisfy \(x^2 = 81\)?
Show solution
We want every number \(x\) whose square is \(81\). Start with the obvious one. Since \(9^2 = 81\), the value \(x = 9\) works. Now remember that a negative times a negative is positive, so squaring a negative number gives a positive result. That means \((-9)^2 = (-9)\times(-9) = 81\), so \(x = -9\) works too. No other number squares to \(81\), because any number larger than \(9\) in size has a square bigger than \(81\), and any number smaller in size has a square smaller than \(81\). So exactly two values, \(9\) and \(-9\), satisfy the equation. This is the heart of the two-answers idea. Solving \(x^2 = 81\) gives a pair, while evaluating the radical \(\sqrt{81}\) gives just the one nonnegative value \(9\). The count of solutions here is \(\boxed{2}\).
Review 7. The equation \(x^2 = 225\) has two solutions, \(15\) and \(-15\). Engineers know \(x\) is negative. What is \(x\)?
Show solution
Start by finding the positive number whose square is \(225\). Since \(15^2 = 15 \times 15 = 225\), we have \(\sqrt{225} = 15\). That means \(x^2 = 225\) has two solutions, \(x = 15\) and \(x = -15\), because both \(15^2 = 225\) and \((-15)^2 = 225\). The radical symbol always points to the nonnegative root, but the equation itself keeps both. Here we are told \(x\) is negative, so we choose the negative one and get \(x = \boxed{-15}\).
Review 8. Squaring any real number gives zero or a positive result, never negative. How many real values of \(x\) satisfy \(x^2 = -16\)?
Show solution
Squaring a real number can never give a negative result. A positive times a positive is positive, a negative times a negative is also positive, and \(0^2 = 0\). So \(x^2\) is always zero or larger for any real \(x\). The equation \(x^2 = -16\) asks for a real number whose square is \(-16\), which is below zero, so nothing can satisfy it. The closest perfect square nearby is \(16\), and \(\sqrt{16} = 4\) shows that \(4^2 = 16\), a positive value, never \(-16\). There are no real solutions, so the count is \(\boxed{0}\).
Review 9. Squaring undoes a square root and vice versa. Evaluate \(\left(\sqrt{47}\right)^2\) without finding the decimal value of \(\sqrt{47}\).
Show solution
By definition, \(\sqrt{47}\) is the nonnegative number whose square is 47. So if you take that number and square it, you must get 47 right back. You can confirm the pattern on a perfect square, where \(\sqrt{49} = 7\) and \(7^2 = 49\), and the same cancellation works for any nonnegative number, including 47. That gives \(\left(\sqrt{47}\right)^2 = 47\), so the answer is \(\boxed{47}\).
Review 10. A recorded side length of \(-8\) has the wrong sign. Evaluate \(\sqrt{(-8)^2}\) to recover the nonnegative side length.
Show solution
Start inside the radical and square the recorded number. Since a negative times a negative is positive, \((-8)^2 = 64\). Now the expression is \(\sqrt{64}\). The square root asks for the nonnegative number whose square is \(64\), and \(8 \times 8 = 64\), so \(\sqrt{64} = 8\). Notice what happened. Squaring turned \(-8\) into \(64\), and the radical brought back the nonnegative version, \(8\), not the original \(-8\). That is why \(\sqrt{x^2}\) gives the nonnegative size, here \(\boxed{8}\).
Review 11. A square hall uses \(4{,}900\) tiles. Since \(4{,}900 = 49 \times 100\), use \(\sqrt{4{,}900} = \sqrt{49} \times \sqrt{100}\) to find the number of tiles along one edge.
Show solution
A square root spreads across a product, so \(\sqrt{49 \times 100} = \sqrt{49} \times \sqrt{100}\). Each piece is a perfect square you can read off directly, with \(\sqrt{49} = 7\) and \(\sqrt{100} = 10\). Multiplying gives \(7 \times 10 = 70\). Check it by squaring, since \(70^2 = 4{,}900\) matches the total number of tiles, so one edge of the hall is \(70\) tiles long. \(\boxed{70}\)
Review 12. A friend claims \(\sqrt{289} = 17\). Verify: does \(17^2 = 289\)? What is \(\sqrt{289}\)?
Show solution
Both sanity checks point the same way, so I confirm them with a direct computation. The last-digit check says the root must end in \(3\) or \(7\), and \(17\) ends in \(7\), so it passes. The size check says squaring the candidate should land exactly on \(289\). Squaring gives \(17^2 = 17 \times 17 = 289\), which is an exact hit. A nearby wrong guess like \(19\) would fail, since \(19\) ends in \(9\) and \(19^2 = 361\) overshoots \(289\). So the root that squares back to \(289\) is \(\sqrt{289} = \boxed{17}\).
Review 13. You know \(\sqrt{x} = 19\). Square both sides to find \(x\). What is \(x\)?
Show solution
The equation \(\sqrt{x} = 19\) keeps \(x\) trapped under the radical, and squaring is the move that undoes a square root, so square both sides. On the left, squaring the root gives back \(x\), and on the right \(19\) becomes \(19^2\), so \(x = 19^2 = 361\). To be sure, check the original equation, \(\sqrt{361} = 19\) since \(19 \times 19 = 361\), which matches. So \(x = \boxed{361}\).
Review 14. A game requires solving \(\sqrt{x + 7} = -3\). The radical is always nonnegative. How many solutions exist?
Show solution
The left side is \(\sqrt{x + 7}\), and the radical symbol always returns the principal root, which is zero or positive. For example \(\sqrt{16} = 4\) and \(\sqrt{49} = 7\), never a negative value. The right side is \(-3\), a negative number. A quantity that is always nonnegative can never equal a negative number, so no choice of \(x\) makes the two sides match. If you tried to solve by squaring both sides you would get \(x + 7 = 9\), which gives \(x = 2\), but checking it shows \(\sqrt{2 + 7} = \sqrt{9} = 3\), not \(-3\). That candidate fails, which confirms there is no real solution. The number of solutions is \(\boxed{0}\).
