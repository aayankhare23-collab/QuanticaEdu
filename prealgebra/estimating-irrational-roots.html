Prealgebra · Lesson 8.2

Estimating Irrational Roots

Solve this lesson — free →All lessons

In 8.1 every radicand was a perfect square. Now suppose a tile setter needs a square panel covering exactly \(50\) square inches. The side is \(\sqrt{50}\), an irrational number whose decimal never ends or repeats, so we estimate it instead.

Key idea
Irrational roots. When the number under the radical is not a perfect square, its root is a new kind of number. It cannot be written as a fraction, and its decimal runs forever without ever ending or settling into a repeating pattern, so there is no exact value to type. But you are not stuck, because the perfect squares from 8.1 sit at fixed, known spots on the number line. Find the two perfect squares that bracket the radicand, and you trap one of these endless roots tightly between two whole numbers you know exactly.
Fun fact
The Pythagoreans believed every length was a fraction. When someone proved \(\sqrt{2}\) cannot be written as any fraction, legend says the discoverer was thrown overboard for shattering that belief.
A square patch covers \(54\) sq m, so the side is \(\sqrt{54}\). Since \(7^2 = 49\) and \(8^2 = 64\), the root lies between \(7\) and \(8\). What is the largest whole number less than \(\sqrt{54}\)?
Show solution
The floor of \(\sqrt{54}\) is the largest whole number whose square does not pass \(54\). Compare exact squares around \(54\). The whole number \(7\) gives \(7^2 = 49\), and \(49 \le 54\), so \(7\) is safe. The next whole number \(8\) gives \(8^2 = 64\), and \(64 > 54\), so \(8\) is too big. Putting those together, \(49 \le 54 < 64\). Because a larger number always has a larger square root, that same order carries over to the roots, giving \(7 < \sqrt{54} < 8\). The side length lands strictly between \(7\) and \(8\) meters, so the largest whole number still below it is \(\boxed{7}\).
Key idea
Bounding a root. A bigger number always has a bigger square root, so the moment you find the perfect square just below your number and the one just above it, you have caught the root between two neighboring whole numbers. The smaller of those whole numbers is the floor of the root, the largest whole number still below it. So \(\sqrt{54}\) is trapped between \(7\) and \(8\), since \(49\) sits just below \(54\) and \(64\) just above, which makes its floor \(7\).
Key idea
Order rule. For numbers that are zero or bigger, squaring never flips the order around, so a larger number always carries a larger square root. That means \(\sqrt{a} > \sqrt{b}\) exactly when \(a > b\), and comparing two roots becomes the easy job of comparing the plain numbers hiding inside them. So \(\sqrt{53} > \sqrt{46}\) without any estimating, just because \(53 > 46\). This one rule is the engine behind trapping a root and behind everything else in the lesson.
A skylight has area \(83\) sq in. The edge \(\sqrt{83}\) is between \(9\) and \(10\). The halfway square is \(9.5^2 = 90.25\). To the nearest inch, what is \(\sqrt{83}\)?
Show solution
First trap the root between landmarks. Since \(9^2 = 81\) and \(10^2 = 100\), and \(81 \le 83 < 100\), the root \(\sqrt{83}\) lies between \(9\) and \(10\). To round to the nearest whole number, compare the root to the halfway point \(9.5\). Squaring keeps the order for nonnegative numbers, so instead of estimating \(\sqrt{83}\) with a long decimal, I compare \(83\) to \(9.5^2\). To stay exact, write \(9.5 = \frac{19}{2}\), so \(9.5^2 = \frac{19^2}{4} = \frac{361}{4}\). Clear the fraction by comparing \(4 \times 83 = 332\) against \(19^2 = 361\). Since \(332 < 361\), we have \(83 < 9.5^2\), which means \(\sqrt{83} < 9.5\). The root sits below the halfway mark, so it rounds down. To the nearest whole inch, \(\sqrt{83} \approx \boxed{9}\).
A skylight has area \(21\) sq in. The edge \(\sqrt{21}\) is between \(4\) and \(5\). The halfway square is \(4.5^2 = 20.25\). To the nearest inch, what is \(\sqrt{21}\)?
Show solution
First trap the root between whole numbers. Because \(4^2 = 16\) and \(5^2 = 25\), and \(16 < 21 < 25\), the edge \(\sqrt{21}\) lies between \(4\) and \(5\). To round to the nearest whole number, the only question is which side of the midpoint \(4.5\) it falls on. Instead of estimating a long decimal, square the midpoint and compare exactly. The midpoint squared is \(4.5^2 = 20.25\). Now compare the number under the root with that boundary. Since \(21 > 20.25\), squaring tells us \(\sqrt{21} > \sqrt{20.25} = 4.5\). The root has passed the halfway mark, so it rounds up to \(5\). The tempting guess of \(4\) was off, and the squared-boundary test settles it cleanly, giving \(\sqrt{21} \approx 5\) to the nearest whole inch, so the answer is \(\boxed{5}\).
Key idea
Rounding a root. To round \(\sqrt{n}\) to the nearest whole number, you never have to chase its endless decimal, only find the boundary it must beat. The cutoff sits exactly halfway between the two integers the root is trapped between, so square that halfway value \(m.5\) and compare it to \(n\). If \(n > (m.5)^2\) the root rounds up, otherwise it rounds down. The whole decision is made with exact whole-number arithmetic, and no decimal for the root is ever needed.
A pendulum is \(\sqrt{23}\) ft long. The root is between \(4.7\) and \(4.8\). Does it pass the halfway square \(4.75^2 = 22.5625\)? To the nearest tenth, what is \(\sqrt{23}\)?
Show solution
First trap \(\sqrt{23}\) between two tenths by comparing squares. Squaring \(4.7\) and \(4.8\) exactly gives \(4.7^2 = 22.09\) and \(4.8^2 = 23.04\), and since \(22.09 < 23 < 23.04\), the root lies between \(4.7\) and \(4.8\). To round to the nearest tenth, check which side of the midpoint \(4.75\) the root sits on, which means comparing \(23\) with \(4.75^2\). Square the midpoint exactly, getting \(4.75^2 = 22.5625\). To compare without trusting a long decimal, scale by \(10000\), so \(4.75^2\) becomes \(475^2 = 225{,}625\) and \(23\) becomes \(230{,}000\). Since \(225{,}625 < 230{,}000\), we have \(4.75^2 < 23\), so \(4.75 < \sqrt{23}\) and the root is past the midpoint, landing on the \(4.8\) side. Rounded to the nearest tenth, \(\sqrt{23} = \boxed{4.8}\).
A drone hovers at \(\sqrt{41}\) m. The root is between \(6.4\) and \(6.5\). Does it pass the halfway square \(6.45^2 = 41.6025\)? To the nearest tenth, what is \(\sqrt{41}\)?
Show solution
The number \(41\) is not a perfect square, so \(\sqrt{41}\) is irrational and we estimate it. First trap it between consecutive tenths using exact squares. Comparing without decimals, \(6.4^2 = \frac{64^2}{100} = \frac{4096}{100}\), and \(41 = \frac{4100}{100}\), so since \(4096 < 4100\) the root is above \(6.4\). Likewise \(6.5^2 = \frac{4225}{100}\), and since \(4100 < 4225\) the root is below \(6.5\). So \(\sqrt{41}\) lies between \(6.4\) and \(6.5\). To round, square the midpoint \(6.45\) and compare exactly. Squaring \(645\) gives \(645^2 = 416{,}025\), while \(41 \times 10{,}000 = 410{,}000\). Since \(410{,}000 < 416{,}025\), we have \(41 < 6.45^2\), which means \(\sqrt{41} < 6.45\). Sitting below the halfway point, the root rounds down. To the nearest tenth, \(\sqrt{41} \approx \boxed{6.4}\).
A square solar farm covers \(6800\) sq m. Given \(82^2 = 6724\) and \(83^2 = 6889\), what is the largest whole number of meters less than \(\sqrt{6800}\)?
Show solution
I want the two consecutive perfect squares that bracket \(6800\), because for nonnegative numbers a bigger number always has a bigger square root. Testing the nearby whole numbers gives the exact comparison \(82^2 = 6724\) and \(83^2 = 6889\). Since \(6724 \le 6800\) and \(6800 \le 6889\), I have \(6724 \le 6800 \le 6889\). Taking square roots keeps that order, so \(82 < \sqrt{6800} < 83\). The root is strictly above \(82\) because \(6800\) is more than \(6724\), and strictly below \(83\) because \(6800\) is less than \(6889\). The largest whole number that still stays below \(\sqrt{6800}\) is therefore \(\boxed{82}\).
Key idea
Finding the floor. To find the largest whole number below \(\sqrt{n}\), search for consecutive whole numbers \(m\) and \(m+1\) with \(m^2 \le n < (m+1)^2\), then \(m\) is the floor of the root. You can hunt for the two perfect squares that hug \(n\), or square whole numbers going up until you pass it and back off by one. It never breaks because you are only ever comparing squares, and squares stay exact even when the root runs on forever, so you never have to pin down the messy decimal at all.
How many whole numbers \(n\) satisfy \(7 < \sqrt{n} < 11\)? Square the bounds: \(n\) must satisfy \(49 < n < 121\). Count the whole numbers strictly between \(49\) and \(121\).
Show solution
The cleanest move is to turn the root bounds into plain number bounds by squaring. That is fair because for nonnegative numbers, the larger number always has the larger square root, so squaring keeps the order intact. Start with \(\sqrt{n} > 7\). Squaring both sides gives \(n > 7^2\), which is \(n > 49\). Then \(\sqrt{n} < 11\) gives \(n < 11^2\), which is \(n < 121\). So \(n\) is a whole number with \(49 < n < 121\). Both ends are strict. The number \(49\) is excluded because \(\sqrt{49} = 7\) exactly, not greater than \(7\), and \(121\) is excluded because \(\sqrt{121} = 11\) exactly, not less than \(11\). The qualifying whole numbers run from \(50\) up through \(120\). Counting a solid run of whole numbers is last minus first plus one, so \(120 - 50 + 1 = 71\). \(\boxed{71}\)
Key idea
Counting roots in a range. Asking how many whole numbers have a square root between two values is hard to picture while the roots are wrapped in radicals, so square both ends and turn it into a question about the plain numbers underneath. To count roots landing between \(5\) and \(6\), square to ask which whole numbers fall between \(25\) and \(36\), then count on sight. Squaring is fair because a bigger number always has a bigger root, so it never sneaks a value in or out. The only thing left to watch is whether each endpoint is counted or skipped.
How many whole numbers \(k\) satisfy \(\sqrt{130} < k < \sqrt{320}\)? The smallest valid \(k\) is \(12\) and the largest is \(17\). Count them.
Show solution
A whole number \(k\) lands strictly between the two roots exactly when \(130 \lt k^2 \lt 320\), so instead of estimating the roots I compare the squares directly. Walking up the perfect squares, \(11^2 = 121\) is below \(130\) but \(12^2 = 144\) is above \(130\), so the smallest \(k\) that works is \(12\). Going the other way, \(17^2 = 289\) is below \(320\) but \(18^2 = 324\) is above \(320\), so the largest \(k\) that works is \(17\). The valid values are the whole numbers \(12, 13, 14, 15, 16, 17\). Counting them, or computing \(17 - 12 + 1\), gives \(\boxed{6}\).
Key idea
Whole numbers between two roots. Counting the whole numbers caught between two different roots is the trapping trick run twice. Since squaring keeps order for nonnegative numbers, asking which whole numbers \(k\) satisfy \(\sqrt{a} < k < \sqrt{b}\) is the same as asking which ones, when squared, land strictly between \(a\) and \(b\). Bracket each root by itself, read off the smallest whole number above \(\sqrt{a}\) and the largest below \(\sqrt{b}\), then count the run. The order rule keeps the count lined up so none slip past.
Cable A is \(6\sqrt{7}\) m, Cable B is \(4\sqrt{15}\) m. Square each to compare. Option 1 is Cable A, Option 2 is Cable B. Which is longer?
Show solution
Both lengths are positive, and for nonnegative numbers a larger value always has a larger square, so squaring the two lengths lets us compare them without ever touching the roots. Squaring Cable A, \((6\sqrt{7})^2 = 6^2 \times 7 = 36 \times 7 = 252\). Squaring Cable B, \((4\sqrt{15})^2 = 4^2 \times 15 = 16 \times 15 = 240\). These are exact whole numbers, no rounding needed, and \(252 > 240\). Since squaring keeps positive numbers in the same order, the cable with the larger square is the longer one, so Cable A wins. The longer cable is option \(\boxed{1}\).
Key idea
Comparing root expressions. Two nonnegative quantities keep their order when you square them, so whichever one squares to the bigger number is the bigger quantity. To compare any two root expressions, square both and rank the plain whole numbers you get. A coefficient out front comes along for the ride, because it squares too, so \(6\sqrt{7}\) squares to \(6^2 \times 7 = 252\), turning an awkward irrational into one tidy number. The decimals you were reaching for are never needed, and this only decides which is larger, it does not simplify or combine the radicals.
Evaluate \(\left(\sqrt{6}\right)^4\). Group the four factors into two pairs. Each pair \(\sqrt{6} \times \sqrt{6} = 6\). What is the result?
Show solution
The trick is to never touch the decimal value of \(\sqrt{6}\) at all. Group the four copies into pairs. One pair squares the root exactly, \(\left(\sqrt{6}\right)^2 = 6\), since a square and a square root cancel for nonnegative numbers. The fourth power is just that result squared again, so \(\left(\sqrt{6}\right)^4 = \left(\left(\sqrt{6}\right)^2\right)^2 = 6^2\), and \(6^2 = 36\). You can sanity check this against the bracketing idea. We know \(2 < \sqrt{6} < 3\) because the perfect squares \(2^2 = 4\) and \(3^2 = 9\) trap \(6\) between them. Raising everything to the fourth power keeps the order, so \(2^4 < \left(\sqrt{6}\right)^4 < 3^4\), meaning the answer must land between \(16\) and \(81\). The exact value sits comfortably inside that window. So \(\left(\sqrt{6}\right)^4 = \boxed{36}\).
Key idea
Even powers of roots. A square root and the number under it are tied together by one promise, that the root times itself gives back the inside, so \(\sqrt{a} \cdot \sqrt{a} = a\) even though \(\sqrt{a}\) by itself may never settle into a tidy decimal. Any even power of a root is just a stack of those pairs, and each pair collapses to the inside, so the irrational vanishes and a whole number is left. Thus \((\sqrt{a})^4 = a^2\) and in general \((\sqrt{a})^{2k} = a^k\), since splitting an even exponent into pairs raises the inside to half the exponent.
A hiker covers two trails: \(\sqrt{18}\) miles then \(\sqrt{33}\) miles. What is the largest whole number less than \(\sqrt{18} + \sqrt{33}\)?
Show solution
The total length is \(\sqrt{18} + \sqrt{33}\), and I want the largest whole number below it. Squaring is the honest tool here, since for nonnegative numbers a bigger square means a bigger root. First a rough trap. From \(4^2 = 16 < 18 < 25 = 5^2\) I get \(4 < \sqrt{18} < 5\), and from \(5^2 = 25 < 33 < 36 = 6^2\) I get \(5 < \sqrt{33} < 6\). Together that says the sum is between \(9\) and \(11\), too loose to name one floor. Now sharpen with tenths, comparing exact squares each time. Since \(4.2^2 = 17.64 < 18\), I know \(\sqrt{18} > 4.2\). Since \(5.7^2 = 32.49 < 33\), I know \(\sqrt{33} > 5.7\). Adding these safe lower edges, \(\sqrt{18} + \sqrt{33} > 4.2 + 5.7 = 9.9\), which is already above \(9\). For the ceiling, \(4.25^2 = 18.0625 > 18\) gives \(\sqrt{18} < 4.25\), and \(5.75^2 = 33.0625 > 33\) gives \(\sqrt{33} < 5.75\), so the sum is less than \(4.25 + 5.75 = 10\). The total is squeezed into \(9.9 < \sqrt{18} + \sqrt{33} < 10\), comfortably between \(9\) and \(10\). The largest whole number below it is \(\boxed{9}\).
A drone flies \(\sqrt{40}\) mi north then \(\sqrt{75}\) mi east. The total is between \(14\) and \(16\). Does \(\sqrt{40} + \sqrt{75}\) exceed \(15\)? What is the largest whole number below the total?
Show solution
The sum clearly beats \(14\), since \(\sqrt{40} > 6\) and \(\sqrt{75} > 8\) force \(\sqrt{40} + \sqrt{75} > 14\). The real contest is whether it stays under \(15\), so test \(15\) exactly. Ask whether \(\sqrt{40} + \sqrt{75} < 15\). Move one root to the other side, so the question becomes whether \(\sqrt{40} < 15 - \sqrt{75}\). The right side is positive because \(\sqrt{75} < 9 < 15\), so both sides are nonnegative and squaring keeps the order. Squaring gives \(40 < 225 - 30\sqrt{75} + 75\), and the right side collects to \(300 - 30\sqrt{75}\). Rearranging, \(30\sqrt{75} < 260\). One root still remains, so square once more to reach plain whole numbers. The left side squares to \(30^2 \cdot 75 = 900 \cdot 75 = 67{,}500\), and the right side squares to \(260^2 = 67{,}600\). Since \(67{,}500 < 67{,}600\), the comparison \(30\sqrt{75} < 260\) is true, every step reverses cleanly, and the sum really is below \(15\). It already cleared \(14\), so the sum sits strictly between \(14\) and \(15\). The largest whole number less than \(\sqrt{40} + \sqrt{75}\) is \(\boxed{14}\).

Review problems

Review 1. A square sticker covers \(34\) sq cm. Since \(5^2 = 25\) and \(6^2 = 36\), the side \(\sqrt{34}\) is between \(5\) and \(6\). What is the largest whole number less than \(\sqrt{34}\)?
Show solution
To find the floor of \(\sqrt{34}\), hunt for the consecutive whole numbers whose squares bracket \(34\). Squaring \(5\) gives \(5^2 = 25\), which is below \(34\). Squaring \(6\) gives \(6^2 = 36\), which is above \(34\). So \(25 < 34 < 36\), and because a larger nonnegative number always has a larger square root, taking the root of each part keeps the order, giving \(5 < \sqrt{34} < 6\). The root lands strictly between \(5\) and \(6\), so the largest whole number less than \(\sqrt{34}\) is \(\boxed{5}\).
Review 2. Rope 1 is \(\sqrt{63}\) m and Rope 2 is \(\sqrt{57}\) m. Both lie between \(7\) and \(8\). Use the order rule to decide: Option 1 is \(\sqrt{63}\), Option 2 is \(\sqrt{57}\). Which is longer?
Show solution
Squaring a nonnegative number keeps the order, so to compare \(\sqrt{63}\) and \(\sqrt{57}\) I compare their squares. Squaring undoes each root, giving \(63\) and \(57\). Since \(63 > 57\), the root of \(63\) is the larger value, which means \(\sqrt{63} > \sqrt{57}\). Both roots round down to \(7\), so the floors tie, but the exact squares break the tie cleanly. Rope 1 is longer, so the answer is option \(\boxed{1}\).
Review 3. A square window has area \(75\) sq in. The edge \(\sqrt{75}\) is between \(8\) and \(9\). Halfway square: \(8.5^2 = 72.25\). To the nearest inch, what is \(\sqrt{75}\)?
Show solution
The side of the window is \(\sqrt{75}\), and you want it to the nearest whole inch. First trap it between landmarks. Since \(8^2 = 64\) and \(9^2 = 81\), and \(75\) sits between \(64\) and \(81\), you know \(8 < \sqrt{75} < 9\). To round, you compare \(\sqrt{75}\) to the halfway value \(8.5\). Rather than work with the decimal square \(72.25\), keep everything as whole numbers by doubling. Doubling the root gives \(2\sqrt{75}\), whose square is \(4 \times 75 = 300\). Doubling \(8.5\) gives \(17\), whose square is \(289\). Since \(300 > 289\), the root \(\sqrt{75}\) is greater than \(8.5\), so it rounds up to \(9\) inches. To the nearest whole inch, \(\sqrt{75} = \boxed{9}\).
Review 4. A ramp rises \(\sqrt{44}\) feet. Since \(6^2 = 36\) and \(7^2 = 49\), the height is between \(6\) and \(7\). The halfway square is \(6.5^2 = 42.25\). To the nearest whole foot, what is \(\sqrt{44}\)?
Show solution
Start by trapping the root between landmark perfect squares. Because \(6^2 = 36\) and \(7^2 = 49\), and 44 sits between 36 and 49, the order rule tells us \(6 < \sqrt{44} < 7\). So the height is somewhere in the 6 to 7 range, and rounding comes down to which integer is nearer. The tie breaker is the midpoint 6.5, and squaring it gives \(6.5^2 = 42.25\). Compare the number under the root with that midpoint square. Since \(44 \ge 42.25\), the value 44 sits above the halfway mark, which means \(\sqrt{44}\) is greater than 6.5 and therefore closer to 7 than to 6. To keep the comparison fully exact you can clear the decimal by doubling before squaring, since \(\sqrt{44}\) versus \(6.5\) is the same as \(2\sqrt{44}\) versus \(13\), and squaring both nonnegative sides gives \(4 \times 44 = 176\) against \(13^2 = 169\). Because \(176 \ge 169\), the root really is past the midpoint, confirming it rounds up. To the nearest whole foot the ramp height is \(\boxed{7}\).
Review 5. A wire is \(\sqrt{31}\) inches long. The root is between \(5.5\) and \(5.6\). Halfway square: \(5.55^2 = 30.8025\). To the nearest tenth, what is \(\sqrt{31}\)?
Show solution
Squaring keeps order for nonnegative numbers, so I can compare squares instead of roots. The consecutive tenths around the answer are \(5.5\) and \(5.6\). Their squares are \(5.5^2 = 30.25\) and \(5.6^2 = 31.36\). Since \(30.25 \le 31 \le 31.36\), the value \(\sqrt{31}\) lies between \(5.5\) and \(5.6\). To round, I test the halfway point \(5.55\), where \(5.55^2 = 30.8025\). Because \(30.8025 \le 31\), the root \(\sqrt{31}\) is greater than \(5.55\), which means it sits in the upper part of the interval and rounds up. To the nearest tenth, \(\sqrt{31} = \boxed{5.6}\).
Review 6. A flagpole is \(\sqrt{88}\) units tall. The root is between \(9.3\) and \(9.4\). Halfway square: \(9.35^2 = 87.4225\). To the nearest tenth, what is \(\sqrt{88}\)?
Show solution
Start by trapping \(\sqrt{88}\) between two tenths. Squaring gives \(9.3^2 = 86.49\) and \(9.4^2 = 88.36\), and since \(86.49 \le 88 \le 88.36\), the root lies between \(9.3\) and \(9.4\). To round, compare \(88\) with the square of the midpoint \(9.35\). Working exactly, \(935^2 = 874225\) while \(88 \times 10000 = 880000\), and \(880000 > 874225\), so \(9.35^2 = 87.4225\) is less than \(88\). That means \(\sqrt{88}\) is larger than \(9.35\), past the halfway mark, so it rounds up to the next tenth. To the nearest tenth, \(\sqrt{88} \approx \boxed{9.4}\).
Review 7. A square field covers \(250\) sq yd. Since \(15^2 = 225\) and \(16^2 = 256\), the side \(\sqrt{250}\) is between \(15\) and \(16\). What is the largest whole number less than \(\sqrt{250}\)?
Show solution
I want the largest whole number that is still smaller than \(\sqrt{250}\), so I trap the root between two consecutive integers by comparing exact squares. The perfect square just below \(250\) is \(15^2 = 225\), and the next one up is \(16^2 = 256\). Since \(225 \le 250 \le 256\) and a larger number always has a larger square root for nonnegative numbers, I get \(15 < \sqrt{250} < 16\). The root is bigger than \(15\) (because \(250\) is more than \(225\)) but it has not reached \(16\) yet (because \(250\) is less than \(256\)). So the largest whole number below \(\sqrt{250}\) is \(\boxed{15}\).
Review 8. A square reservoir has footprint \(9000\) sq m. Find the largest whole number less than \(\sqrt{9000}\).
Show solution
The largest whole number less than \(\sqrt{9000}\) is the floor of that root, so I look for the two consecutive whole numbers whose squares trap \(9000\). Testing nearby values, \(94^2 = 8836\) and \(95^2 = 9025\). Since \(8836 < 9000 < 9025\), squaring shows \(94 < \sqrt{9000} < 95\). Because squaring preserves order for nonnegative numbers, \(\sqrt{9000}\) lands strictly between \(94\) and \(95\), so the largest whole number below it is \(\boxed{94}\).
Review 9. Find the largest whole number less than \(\sqrt{3{,}000{,}000}\). Since the root is in the seventeen-hundreds, test \(1{,}732^2\) and \(1{,}733^2\).
Show solution
I want the largest whole number whose square stays at or below \(3{,}000{,}000\), because that whole number is the floor of \(\sqrt{3{,}000{,}000}\). Squaring keeps order for nonnegative numbers, so I compare exact squares instead of decimals. Testing \(1{,}732\) gives \(1{,}732^2 = 2{,}999{,}824\), which is at or below \(3{,}000{,}000\). Testing the next number up, \(1{,}733^2 = 3{,}003{,}289\), which passes \(3{,}000{,}000\). So \(1{,}732^2 \le 3{,}000{,}000 < 1{,}733^2\), which traps the root between \(1{,}732\) and \(1{,}733\). The largest whole number less than \(\sqrt{3{,}000{,}000}\) is \(\boxed{1732}\).
Review 10. How many whole numbers \(n\) satisfy \(\sqrt{50} < \sqrt{n} < \sqrt{130}\)? Square the bounds and count whole numbers strictly between \(50\) and \(130\).
Show solution
The order rule from this lesson says that for nonnegative numbers, the one with the larger value has the larger square root, and the comparison runs both ways. So \(\sqrt{50} < \sqrt{n}\) is the same as \(50 < n\), and \(\sqrt{n} < \sqrt{130}\) is the same as \(n < 130\). I squared each given root to get an exact bound, with \(\left(\sqrt{50}\right)^2 = 50\) and \(\left(\sqrt{130}\right)^2 = 130\). Now I just need the whole numbers \(n\) with \(50 < n < 130\). Since both ends are left out, the smallest is \(51\) and the largest is \(129\). Counting from \(51\) through \(129\) gives \(129 - 51 + 1 = 79\) whole numbers. The count is \(\boxed{79}\).
Review 11. How many whole numbers \(k\) satisfy \(\sqrt{40} < k < \sqrt{210}\)? Find the smallest whole number above \(\sqrt{40}\) and the largest below \(\sqrt{210}\), then count.
Show solution
Instead of estimating the roots as decimals, compare squares, since for nonnegative numbers a bigger square means a bigger root. For the left post, \(6^2 = 36\) is less than 40 and \(7^2 = 49\) is more than 40, so \(6 < \sqrt{40} < 7\). The smallest whole number strictly above \(\sqrt{40}\) is 7. For the right post, \(14^2 = 196\) is less than 210 and \(15^2 = 225\) is more than 210, so \(14 < \sqrt{210} < 15\). The largest whole number strictly below \(\sqrt{210}\) is 14. The fenced-in whole numbers run from 7 through 14, namely 7, 8, 9, 10, 11, 12, 13, 14. Counting them gives \(14 - 7 + 1 = 8\). So \(\boxed{8}\).
Review 12. Beam 1 is \(7\sqrt{5}\) m and Beam 2 is \(4\sqrt{15}\) m. Square each to compare. Option 1 is Beam 1, Option 2 is Beam 2. Which is longer?
Show solution
Squaring a positive length keeps the order, so I can compare the squares instead of the messy roots. Beam 1 gives \(\left(7\sqrt{5}\right)^2 = 7^2 \times 5 = 49 \times 5 = 245\). Beam 2 gives \(\left(4\sqrt{15}\right)^2 = 4^2 \times 15 = 16 \times 15 = 240\). Since \(245 \ge 240\), Beam 1 has the larger square, so Beam 1 is the longer beam. The answer is option \(\boxed{1}\).
Review 13. Evaluate \(\sqrt{3}^6\). Group the six factors into three pairs; each pair \(\sqrt{3} \cdot \sqrt{3} = 3\). What whole number is the result?
Show solution
An even power of a root always escapes back to a whole number, because two copies of a root undo it. Write the sixth power as three squared pieces, \(\sqrt{3}^6 = \left(\sqrt{3}^2\right)^3\). Each \(\sqrt{3}^2\) is just \(3\), since squaring a square root returns the number underneath. That leaves \(3^3 = 3 \cdot 3 \cdot 3 = 27\). You can sanity check by comparing exact squares the other way, where the answer \(27\) squared is \(729\), and \(\sqrt{3}^6\) squared is \(\sqrt{3}^{12} = 3^6 = 729\) as well, so the two truly match. The irrational root vanished and a clean integer is left, \(\boxed{27}\).
Review 14. Two trails are \(\sqrt{28}\) miles and \(\sqrt{52}\) miles long. What is the largest whole number less than \(\sqrt{28} + \sqrt{52}\)?
Show solution
I will trap each root between tenths by squaring, since for nonnegative numbers a larger square means a larger root. For the first trail, \(5.2^2 = 27.04\) which is below \(28\), and \(5.3^2 = 28.09\) which is above \(28\), so \(5.2 < \sqrt{28} < 5.3\). For the second trail, \(7.2^2 = 51.84\) which is below \(52\), and \(7.3^2 = 53.29\) which is above \(52\), so \(7.2 < \sqrt{52} < 7.3\). Adding the low ends, the total is more than \(5.2 + 7.2 = 12.4\), and adding the high ends, the total is less than \(5.3 + 7.3 = 12.6\). So the sum sits between \(12.4\) and \(12.6\), comfortably between \(12\) and \(13\). Because \(\sqrt{28} + \sqrt{52}\) is irrational it never lands exactly on a whole number, so the largest whole number below it is \(\boxed{12}\).
