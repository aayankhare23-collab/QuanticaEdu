Prealgebra · Lesson 6.5

Inequalities

Solve this lesson — free →All lessons

Every equation so far ended on one exact value, the single number that makes both sides match. Real life is rarely that tidy. A recipe wants the oven at least \(350\) degrees, so \(351\) and \(400\) work too, and a bridge bars trucks above a posted height, so countless heights pass. Each answer is a whole stretch of the number line, called an inequality. You already own the tools: the same balance moves from 6.1 through 6.4, with exactly one new twist waiting near the end.

A solo ride only lets you board if your ticket count is strictly more than \(5\), written \(t > 5\). The word strictly matters, because holding exactly \(5\) is not enough, you have to clear the line, not land on it. Five friends hold \(3\), \(5\), \(7\), \(12\), and \(5.5\) tickets. How many of those counts satisfy \(t > 5\)? Type the single number.
Show solution
The rule is already solved for \(t\), so there is no rearranging to do and no sign to flip. The condition is simply \(t > 5\), which in words means the ticket count must be larger than \(5\), and the strict symbol \(>\) means \(5\) itself is left out. Now test each friend against \(t > 5\). For \(t = 3\), is \(3 > 5\)? No. For \(t = 5\), is \(5 > 5\)? No, because \(5\) is equal to \(5\), not greater, and strictly more means you cannot sit on the boundary. For \(t = 7\), is \(7 > 5\)? Yes. For \(t = 12\), is \(12 > 5\)? Yes. For \(t = 5.5\), is \(5.5 > 5\)? Yes, since half a ticket past \(5\) still clears the line. The solution range is every value above \(5\), written \(t > 5\), and the friend at exactly \(5\) falls just outside it. Counting the yeses gives \(7\), \(12\), and \(5.5\), which is three friends. So the number allowed to ride alone is \(\boxed{3}\).
Key idea
Inequality basics. An equals sign swears the two sides are the same number, pinning you to one answer, but an inequality only swears one side outranks the other, so a whole band of numbers can make it true at once. Keep four symbols straight: \(>\) and \(<\) are strict, when the sides may never tie, while \(\ge\) and \(\le\) let the boundary itself join. Because a range works instead of one number, the picture is a ray sliding off in one direction, not a single dot.
Key idea
Reading the graph. The little circle at the end is a gate. A hollow open circle means that exact number is locked out, the case for strict \(>\) and \(<\); you can creep as close as you like but never land on it. A filled closed circle swings the gate open and lets the endpoint in, since the or-equal part of \(\ge\) and \(\le\) says equality counts. The arrow finishes the story, carrying every value that works off in its direction.
A delivery cart already holds \(8\) crates, and the running total must stay under \(20\), so with \(x\) more crates the rule is \(x + 8 < 20\). Solve it the same way as \(x + 8 = 20\), by subtracting \(8\) from both sides, with no sign flip for subtraction. The boundary is strict, so a value sitting exactly on it does not count. What is the largest whole number \(x\) can be?
Show solution
Start with the rule from the problem, \(x + 8 < 20\). To get \(x\) by itself, subtract \(8\) from both sides, exactly the move you would make for the equation \(x + 8 = 20\). Subtracting the same amount from both sides shifts the values but never reverses the inequality, so there is no sign flip here and the \(<\) keeps pointing the same way. That leaves \(x < 20 - 8\), which is \(x < 12\). In words, the solution is every value strictly less than \(12\), written \(x < 12\). The bound is strict, so \(12\) is not allowed, you would need fewer than \(12\) more crates. The largest whole number strictly below \(12\) is \(11\). Check it. Loading \(11\) more crates gives \(11 + 8 = 19\), and \(19 < 20\) is true, while \(12\) more would give \(12 + 8 = 20\), and \(20 < 20\) is false, so \(11\) really is the top. The most crates you may stack on is \(\boxed{11}\).
A diver's gauge reads negatively below the surface. Her reading is \(x\), she rises \(5\) meters to \(x + 5\), and the safety check needs the new reading at least \(2\), so \(x + 5 \ge 2\). Subtract \(5\) from both sides; subtracting never flips the sign. Because the sign is \(\ge\), the boundary itself is allowed. What is the smallest value \(x\) can be?
Show solution
Start with the safety condition \(x + 5 \ge 2\). You solve it exactly the way you solved equations all chapter, by undoing operations one at a time. Undo the \(+5\) by subtracting 5 from both sides, which gives \(x \ge -3\). No sign flip happens here, because the sign only reverses when you multiply or divide both sides by a negative, and subtracting a number never does that, so the \(\ge\) keeps pointing the same way. In words, her starting reading can be \(-3\) or anything greater. In symbols that is \(x \ge -3\). Because the sign is \(\ge\) and not \(>\), the boundary \(-3\) is itself allowed, so the gate is closed and \(-3\) is a value she is permitted to start from. The smallest value \(x\) can be is therefore the boundary number, \(-3\). $$\boxed{-3}$$
Four identical sandbags sit on a shelf rated for at most \(26\) kilograms total, so with one bag weighing \(x\), the rule is \(4x \le 26\). Solve it like an equation, keeping the inequality sign: divide both sides by the positive number \(4\), which does not flip the sign. The word most means the boundary counts. What is the largest value \(x\) can be? Give a fraction in lowest terms.
Show solution
Start from the setup \(4x \le 26\). The four bags together cannot top \(26\) kilograms, so their combined weight \(4x\) is at most \(26\). To isolate \(x\), divide both sides by \(4\). Because \(4\) is a positive number, the inequality keeps pointing the same direction, so there is no sign flip here. $$\frac{4x}{4} \le \frac{26}{4}$$ That leaves \(x \le \tfrac{26}{4}\), and \(\tfrac{26}{4}\) reduces to \(\tfrac{13}{2}\). So the solution range is every weight up to and including \(\tfrac{13}{2}\), in words "x is at most thirteen halves" and in symbols \(x \le \tfrac{13}{2}\). The word "most" means the boundary counts, the gate is closed, so \(\tfrac{13}{2}\) itself is allowed. The largest value \(x\) can be is therefore the boundary, \(\boxed{13/2}\) kilograms.
Key idea
The same solving moves carry over. Solving an inequality is the same game as solving an equation, just with a \(>\) or \(<\) where the \(=\) used to be. Tidy each side, undo a plus with a minus, undo a times with a divide, and the bigger side stays the bigger side, because adding or scaling both sides by the same positive amount moves them together. Every move from 6.2 and 6.3 carries straight over. There is exactly one move not tried yet, scaling by a negative, waiting in the next problem.
Fun fact
The symbols \(<\) and \(>\) first appeared in Thomas Harriot's 1631 book. The wide end faces the larger value. The rounded cousins \(\le\) and \(\ge\) were popularized by Pierre Bouguer in the 1700s.
Here is a statement nobody would argue with: \(3 < 8\). Now multiply both sides by \(-1\), so the \(3\) becomes \(-3\) and the \(8\) becomes \(-8\). On a number line, the one farther left is the smaller, so \(-8\) is now the smaller value. Of the two results \(-3\) and \(-8\), which one is the larger? Type that value.
Show solution
Start with a true statement, \(3 < 8\). Multiply both sides by \(-1\). The left side becomes \(-3\) and the right side becomes \(-8\). Here is the surprise. If you just copied the old direction you would write \(-3 < -8\), but that is false, because \(-8\) is farther left on the number line and is the smaller number. To keep the statement true the inequality sign has to flip. This is the sign flip. Multiplying or dividing both sides by a negative reverses the direction of the inequality, so the true statement is \(-3 > -8\). Reflecting both numbers across zero turned the order around. In words, after the flip \(-3\) is now the larger value and \(-8\) is the smaller one, and the relationship is \(-3 > -8\). The question asks for the larger of the two results, which is \(\boxed{-3}\).
Key idea
The one new rule. The moment you multiply or divide both sides by a negative number, you must flip the inequality, so \(<\) becomes \(>\), \(\le\) becomes \(\ge\), and the same the other way, because crossing to the negative side reverses which value is bigger. Adding, subtracting, and scaling by a positive never flip anything, so those moves are always safe. The whole skill comes down to one question at the very end: did I just divide by a negative?
A freezer alarm trips whenever \(-5x\) climbs above \(17\), so the danger zone is \(-5x > 17\). To free \(x\), divide both sides by \(-5\), and dividing by a negative flips the \(>\) to \(<\). What is the largest integer \(x\) can be?
Show solution
Start with \(-5x > 17\). To free \(x\), divide both sides by \(-5\). Here is the one twist that makes inequalities different from equations. Dividing both sides by a negative number reverses the inequality, so the \(>\) flips to \(<\). $$x < \frac{17}{-5} = -\frac{17}{5} = -3.4$$ In words, \(x\) must be smaller than negative three and four tenths. In symbols that is \(x < -\frac{17}{5}\). The bound is strict, so the gate at \(-\frac{17}{5}\) is open and \(-3.4\) itself is shut out. The question asks for the largest integer that fits. Walking left from \(-3.4\) on the number line, the first integer you reach is \(-4\), since \(-3\) sits to the right of \(-3.4\) and is too big. A quick check, \(-5 \cdot (-4) = 20\), and \(20 > 17\) is true, while \(-5 \cdot (-3) = 15\), and \(15 > 17\) is false. The largest integer \(x\) can be is \(\boxed{-4}\).
This one pulls everything together: the variable appears on both sides and there is a group to clear. Solve \(3(2 - x) > 2x + 1\) by distributing, then gathering the variable, and stay alert because one step divides by a negative and forces the sign to flip. Once you have the range, what is the largest integer \(x\) can be?
Show solution
Treat this like any both-sides equation right up until the very last step. First distribute the \(3\) across the parentheses. $$3(2 - x) > 2x + 1 \quad\Rightarrow\quad 6 - 3x > 2x + 1$$ Now gather the variable terms. Subtract \(2x\) from both sides. $$6 - 5x > 1$$ Subtract \(6\) from both sides to isolate the variable term. $$-5x > -5$$ Here is the one move that makes inequalities different. To get \(x\) by itself you divide both sides by \(-5\), and dividing by a negative number FLIPS the inequality sign. The \(>\) becomes \(<\). $$x < 1$$ In words, every number strictly less than \(1\) makes the original statement true, and \(1\) itself does not, because the bound is strict. In symbols the solution is \(x < 1\). The question asks for the largest integer \(x\) can be. Since \(1\) is shut out by the strict sign, the largest integer that still fits is the whole number just below it, which is \(0\). A quick check confirms it. Putting \(x = 0\) into the original gives \(3(2) = 6\) on the left and \(1\) on the right, and \(6 > 1\) is true, while \(x = 1\) gives \(3\) on the left and \(3\) on the right, and \(3 > 3\) is false. The largest integer is \(\boxed{0}\).
The trickiest one, folding everything together. Solve \(5 - 2x - 8 > 4 - x\). The left side has two number pieces sitting apart, so tidy it first, then gather the variables. One move near the end forces the sign to turn around. Once you have the range, what is the largest integer \(x\) can be?
Show solution
Begin by tidying the left side. The numbers \(5\) and \(-8\) combine, and \(5 - 8 = -3\), so the inequality becomes \(-3 - 2x > 4 - x\). Now gather the variables. Add \(x\) to both sides, and \(-2x + x = -x\), which gives \(-3 - x > 4\). Add \(3\) to both sides to clear the number on the left, leaving \(-x > 7\). The last move is to turn \(-x\) into \(x\) by multiplying both sides by \(-1\). Multiplying by a negative reverses the inequality, so the sign flips and the result is \(x < -7\). In words, \(x\) is any number strictly less than negative seven, written \(x < -7\). The gate at \(-7\) is open, so \(-7\) does not count, and the largest integer below it is \(-8\). A quick check confirms it, since putting \(x = -8\) into the original gives \(13 > 12\), which is true, while \(x = -7\) gives \(11 > 11\), which is false. So the largest integer \(x\) can be is \(\boxed{-8}\).
A compound inequality traps the variable in the middle: solve \(1 < 5 - 2x \le 11\). Whatever you do, do it to all three parts at once so the middle stays fenced in on both sides, and remember that dividing by the negative coefficient flips both signs together. Once you have \(x\) cornered, how many integers make the whole thing true? Type the single count.
Show solution
Treat all three parts as one moving system. Start from \(1 < 5 - 2x \le 11\) and subtract \(5\) from every part, which gives \(-4 < -2x \le 6\). Now divide every part by \(-2\). Because \(-2\) is negative, both inequality signs FLIP, and that sign flip is the one twist this whole topic adds. Dividing turns it into \(2 > x \ge -3\). Reading that from small to large, the solution range is \(-3 \le x < 2\), in words every number from \(-3\) included up to \(2\) but not reaching \(2\). The gate at \(-3\) is closed and the gate at \(2\) is open. Quick checks confirm the boundaries. At \(x = -3\), \(5 - 2(-3) = 11\), and \(11 \le 11\) holds, so \(-3\) is in. At \(x = 2\), \(5 - 2(2) = 1\), and \(1 < 1\) is false, so \(2\) is out. The integers in the range are \(-3, -2, -1, 0, 1\), which is \(5\) of them. The number of integers that satisfy the inequality is \(\boxed{5}\).
Key idea
Compound inequalities. Something like \(4 < x \le 9\) is two demands holding at once, a floor underneath and a ceiling overhead, so instead of an endless ray the answer is a fenced-in segment with both ends marked. Solving it is the same balance work, except now there are three parts side by side, so whatever you do to one part you do to the left, middle, and right together. And the negative twist never went away: scaling all three by a negative flips both ends at once.
Priya pays a flat \(\$45\) booth fee and clears \(\$5\) on each eraser she sells, so selling \(n\) erasers gives a profit of \(5n - 45\) dollars. She wants that profit to be at least \(\$200\), so the goal is \(5n - 45 \ge 200\). Solve it, dividing by the positive \(5\) without flipping the sign. What is the fewest whole erasers she must sell to reach her goal?
Show solution
Set up the goal from the profit. Selling \(n\) erasers earns \(5n\) dollars after production cost, and the \(45\) dollar booth fee comes out once, so the profit is \(5n - 45\). She wants it to be at least \(200\), which gives \(5n - 45 \ge 200\). Solve it exactly the way you solved equations in this chapter, with one thing to watch. First add \(45\) to both sides to free the \(n\) term, and that gives \(5n \ge 245\). Now divide both sides by \(5\). Because \(5\) is positive, the inequality sign does not flip, it keeps pointing the same way, and you get \(n \ge 49\). In words, she meets her goal as long as she sells \(49\) or more erasers, written \(n \ge 49\). Check the boundary. At \(n = 49\) the profit is \(5 \cdot 49 - 45 = 245 - 45 = 200\) dollars, which hits the goal exactly since "at least" includes \(200\). At \(n = 48\) the profit is \(5 \cdot 48 - 45 = 240 - 45 = 195\) dollars, which falls short. She cannot sell a fraction of an eraser, and \(49\) is a whole number that already works, so the fewest whole erasers she must sell is \(\boxed{49}\).
Three runners are scored so a bigger number means faster. The sheet gives two facts: Priya beat Quinn, \(P > Q\), and Quinn was at least as fast as Rafe, \(Q \ge R\). A teammate scribbles five claims, \(P > R\), \(P \ge R\), \(P > Q\), \(Q > R\), and \(R > P\). Using only the two facts and the way inequalities chain when they point the same way, how many of the five claims are forced to be true? Type the single count.
Show solution
Start by stacking the two given facts so their inequality signs face the same direction. Since \(P > Q\) and \(Q \ge R\), the whole picture becomes one chain, \(P > Q \ge R\). Reading a chain like this is the heart of the problem. When every link points the same way you may follow them across and combine them, and a strict \(>\) anywhere along the chain keeps the combined result strict. Now test each of the five claims against the chain. The claim \(P > Q\) is one of the two facts we were handed, so it is forced to be true. The claim \(P > R\) asks whether Priya beats Rafe, and following the chain from \(P\) past \(Q\) all the way to \(R\) crosses the strict \(>\) between \(P\) and \(Q\), so the combined comparison stays strict and \(P > R\) is forced. The claim \(P \ge R\) says Priya is at least as fast as Rafe, which is a weaker statement than \(P > R\), and since the stronger \(P > R\) is already guaranteed, \(P \ge R\) is forced too. That is three forced so far. The claim \(Q > R\) is not forced, because the link between \(Q\) and \(R\) is only \(\ge\), so Quinn and Rafe are allowed to tie. If \(Q = R\) then \(Q > R\) is false, which means this claim can fail. The claim \(R > P\) is not forced either, it is actually backwards, since the chain puts \(P\) at the fast end and \(R\) at the slow end, so \(R > P\) is impossible. Stating the result in words, the forced claims are \(P > Q\), \(P > R\), and \(P \ge R\), while \(Q > R\) and \(R > P\) are not forced. Counting the forced claims gives \(3\). \(\boxed{3}\)
Key idea
Chaining inequalities. When comparisons all lean the same way, hook them into one chain and read straight across, so if a first amount tops a second and the second is at least as big as a third, the first must top the third. Watch whether the links are strict: a single strict \(>\) anywhere travels the whole length and the ends come out strictly apart, but an all-\(\ge\) chain cannot promise the ends differ, since a quiet tie could slip through every step.
A metal sample starts at \(22\) degrees Celsius and the cooler pulls it down \(1.5\) degrees per minute, so after \(m\) minutes its temperature is \(22 - 1.5m\). It freezes only once that drops strictly below \(-8\), so the condition is \(22 - 1.5m < -8\). Solve it, remembering that dividing by the negative coefficient flips the sign. What is the fewest whole minutes the lab must wait until it freezes?
Show solution
Set up the freezing condition. After \(m\) minutes the temperature is \(22 - 1.5m\), and the sample freezes once that drops strictly below \(-8\), so the inequality is \(22 - 1.5m < -8\). Solve it the same way you solve any linear equation, one step at a time. Subtract \(22\) from both sides to get \(-1.5m < -30\). Now divide both sides by \(-1.5\). This is the one twist of the whole chapter. Dividing both sides by a negative number flips the inequality, so the \(<\) becomes \(>\) and you get \(m > 20\). In words, the sample is frozen for every number of minutes greater than \(20\), and because the bound is strict, \(20\) itself is not included. The gate at \(20\) is open. Now check the two values right around that gate. At \(m = 20\) the temperature is \(22 - 1.5 \cdot 20 = -8\), which is exactly on the line and not yet below it, so the sample has not frozen. At \(m = 21\) the temperature is \(22 - 1.5 \cdot 21 = -9.5\), which is below \(-8\), so it has frozen. The fewest whole minutes is the first whole number that satisfies \(m > 20\), which is \(\boxed{21}\).

Review problems

Review 1. A rewards account stays open as long as its balance \(w\) satisfies \(w \ge -4\), since a small overdraft is forgiven. Five accounts have balances \(-6\), \(-4\), \(-3\), \(0\), and \(5\). How many satisfy \(w \ge -4\)?
Show solution
The rule keeps an account open when \(w \ge -4\), which in words is every balance at or above \(-4\). There is no negative multiplier here, so nothing flips. I just check each balance. \(-6\) is below \(-4\), so it fails. \(-4\) equals \(-4\), and the \(\ge\) sign includes equality, so it passes. \(-3\), \(0\), and \(5\) are all above \(-4\), so they pass too. The passing balances are \(-4\), \(-3\), \(0\), and \(5\), which is a count of \(\boxed{4}\).
Review 2. A scanner keeps a reading \(m\) only when \(m < 7\) and tosses anything that hits or passes the threshold, so the strict sign leaves the boundary \(7\) out. One shift gives \(2\), \(7\), \(6.9\), \(8\), and \(-1\). How many readings does the scanner keep?
Show solution
The keep rule is the strict inequality \(m < 7\). A strict less-than is true only when the reading sits below \(7\) and never when it equals \(7\), so the boundary value itself is rejected. Test each reading. For \(2\) we have \(2 < 7\), true, kept. For \(7\) we have \(7 < 7\), false, rejected, since a number is not less than itself. For \(6.9\) we have \(6.9 < 7\), true, kept. For \(8\) we have \(8 < 7\), false, rejected. For \(-1\) we have \(-1 < 7\), true, kept. The kept readings are \(2\), \(6.9\), and \(-1\), which describes the range \(m < 7\), every reading strictly below seven. That is \(3\) readings. The single asked value is the count, \(\boxed{3}\).
Review 3. A ferry deck already holds \(13\) crates, and loading \(x\) more must keep the total under \(25\), so \(x + 13 < 25\). Solve it, then answer: what is the largest integer number of extra crates \(x\) can be?
Show solution
Start from \(x + 13 < 25\). Subtract \(13\) from both sides to undo the addition. Because you are subtracting a number and not multiplying or dividing by a negative, there is no sign flip and the inequality keeps pointing the same way. That leaves \(x < 12\). In words, x can be any value strictly less than twelve, written \(x < 12\). The bound is strict, so \(12\) is not allowed, only values below it. The captain loads whole crates, so the largest integer strictly below \(12\) is \(11\). Check it. Loading \(11\) gives \(11 + 13 = 24\), and \(24 < 25\) is true, while loading \(12\) gives \(12 + 13 = 25\), and \(25 < 25\) is false. The largest integer number of extra crates is \(\boxed{11}\).
Review 4. A pilot's dial setting \(x\) must satisfy \(x - 9 \ge -2\). Add \(9\) to both sides, with no flip for addition. Because the sign is \(\ge\), the boundary is allowed. What is the smallest value \(x\) can be?
Show solution
Start with \(x - 9 \ge -2\). Add 9 to both sides to get \(x\) by itself. The left side becomes \(x\) and the right side becomes \(-2 + 9\), which is \(7\). So \(x \ge 7\). Adding a number never flips an inequality, so the \(\ge\) stays pointing the same way and there is no sign flip to worry about. In words, \(x\) can be 7 or any number bigger than 7. In symbols that range is \(x \ge 7\). Because the bound is closed, 7 itself is allowed, so the smallest value \(x\) can be is exactly 7. The smallest value is \(\boxed{7}\).
Review 5. A garden permit allows at most \(28\) liters, and each visit takes \(8\) liters, so over \(x\) visits the rule is \(8x \le 28\). Divide both sides by the positive \(8\), no flip. The word most means the boundary counts. What is the largest value \(x\) can be? Give a fraction in lowest terms.
Show solution
Start from the permit rule \(8x \le 28\). To isolate \(x\), divide both sides by \(8\). The number \(8\) is positive, so the inequality sign does not flip and keeps pointing the same way. That gives \(x \le \frac{28}{8}\). Now simplify \(\frac{28}{8}\) by dividing top and bottom by \(4\), which gives \(\frac{7}{2}\). So the solution range is every \(x\) that is at most seven halves, written \(x \le \frac{7}{2}\). Because the sign is \(\le\) and not a strict \(<\), the boundary \(\frac{7}{2}\) is included in the range, so it is a closed upper bound. The largest value \(x\) can be is that boundary itself, \(\frac{7}{2}\). Check it. At \(x = \frac{7}{2}\) the water taken is \(8 \cdot \frac{7}{2} = 28\) liters, which lands right on the limit and is allowed, while any larger \(x\) would push past \(28\) and break the rule. So the answer is \(\boxed{7/2}\).
Review 6. Cookie packs cost \(\$7\) each, and Priya wants to spend strictly more than \(\$50\) to clear a free-shipping bar, so \(7x > 50\). Solve it; the strict sign means \(50\) itself does not count. What is the smallest integer number of packs she can buy?
Show solution
Start with \(7x > 50\). To undo the multiplication by \(7\), divide both sides by \(7\). Because \(7\) is positive, the inequality direction does not change, so there is no sign flip here. That gives \(x > \frac{50}{7}\). The fraction \(\frac{50}{7}\) is about \(7.14\), so in words \(x\) is any number strictly greater than \(\frac{50}{7}\), written \(x > \frac{50}{7}\). Since the bound is strict, \(x\) cannot equal \(7.14\), and it certainly cannot be \(7\) because \(7 \times 7 = 49\), which is not more than \(50\). The first whole number above \(7.14\) is \(8\), and checking it gives \(7 \times 8 = 56 > 50\), which works. So the smallest integer number of packs is \(\boxed{8}\).
Review 7. A deep-sea sensor's reading \(x\) must satisfy \(-6x < 30\). Divide both sides by \(-6\) and flip the sign because the divisor is negative. What is the smallest integer \(x\) can be?
Show solution
Start with \(-6x < 30\). To free \(x\) you divide both sides by \(-6\). Dividing by a negative flips the inequality, so the \(<\) becomes \(>\), and you get \(x > -5\). In words, \(x\) must be strictly greater than \(-5\). The boundary \(-5\) is not included, so any value bigger than \(-5\) works. Check it. At \(x = -4\) the left side is \(-6 \cdot (-4) = 24\), and \(24 < 30\) is true. At \(x = -5\) the left side is \(30\), and \(30 < 30\) is false, so \(-5\) really is shut out. The integers allowed are \(-4, -3, -2, \dots\), and the smallest of these is \(-4\). So the smallest integer \(x\) can be is \(\boxed{-4}\).
Review 8. A frost alarm trips whenever \(-5x > 12\), where \(x\) is a temperature that runs colder as it drops. Solve it, dividing by \(-5\) and flipping the sign. What is the largest integer value \(x\) can be?
Show solution
Start with \(-5x > 12\). To free \(x\), divide both sides by \(-5\). Dividing by a negative number flips the inequality, so the \(>\) becomes \(<\). That gives \(x < \frac{12}{-5}\), which simplifies to \(x < -\frac{12}{5}\). As a decimal that boundary is \(-2.4\). So the alarm trips for every temperature strictly below \(-2.4\), in words "x is less than negative two and four tenths" and in symbols \(x < -2.4\). The boundary itself does not count, since the sign is strict. The largest integer that sits to the left of \(-2.4\) on the number line is \(-3\), because \(-2\) is bigger than \(-2.4\) and fails, while \(-3\) is smaller and works. Check it. Plugging in \(-3\) gives \(-5 \cdot (-3) = 15\), and \(15 > 12\) is true, while \(-2\) gives \(10\), and \(10 > 12\) is false. So the largest integer \(x\) can be is \(\boxed{-3}\).
Review 9. A vending machine credits Mara once it counts more than \(19\) tokens past a deposit, and \(x\) cups give \(4x - 5\) such tokens, so she needs \(4x - 5 > 19\). Add \(5\), then divide by the positive \(4\), no flip. What is the smallest integer number of cups \(x\) she can buy?
Show solution
Start from \(4x - 5 > 19\) and add \(5\) to both sides, which gives \(4x > 24\). Now divide both sides by \(4\). Because \(4\) is positive, the inequality sign stays the same, so there is no flip here. That leaves \(x > 6\). In words, Mara needs strictly more than six cups, written \(x > 6\). The bound is strict, so six cups land exactly on \(19\) tokens past the deposit, which is not more than \(19\). The smallest whole number greater than \(6\) is \(7\), and seven cups give \(4(7) - 5 = 23\), comfortably past the threshold. So the smallest integer number of cups is \(\boxed{7}\).
Review 10. A drone starts at a battery reading of \(10\) and the lamp drains \(4\) per minute, so after \(x\) minutes the reading is \(10 - 4x\), and the pilot keeps it on while \(10 - 4x \ge -6\). Subtract \(10\), then divide by \(-4\) and flip the sign. What is the largest value \(x\) can be?
Show solution
Start with \(10 - 4x \ge -6\). Subtract \(10\) from both sides to get \(-4x \ge -16\). Now divide both sides by \(-4\). Because you are dividing by a negative number, the inequality sign flips, so \(\ge\) turns into \(\le\), and you get \(x \le 4\). In words, \(x\) can be any value of \(4\) or less, which is a closed upper bound that includes \(4\) itself. The question asks for the largest value \(x\) can be, and since \(4\) is allowed, that largest value is \(\boxed{4}\). Check the boundary. At \(x = 4\) the reading is \(10 - 4\cdot 4 = -6\), which is exactly \(-6\) or above, so it works. At \(x = 5\) the reading is \(10 - 4\cdot 5 = -10\), which is below \(-6\), so it fails. The cutoff sits right at \(4\).
Review 11. One battery mode reads \(7 - x\) and another reads \(2x + 1\) after \(x\) minutes, and the first beats the second when \(7 - x > 2x + 1\). Gather the variable, then divide by a negative and flip the sign. What is the largest integer \(x\) can be?
Show solution
The machinery is the same as solving an equation, with one twist waiting at the end. Begin with \(7 - x > 2x + 1\) and pull the variable to one side by subtracting \(2x\) from both sides, which gives \(7 - 3x > 1\). Subtract \(7\) from both sides to isolate the \(x\) term, leaving \(-3x > -6\). Here is the twist. To free \(x\) you divide both sides by \(-3\), and dividing by a negative number reverses the inequality, so the \(>\) flips to \(<\) and you get \(x < 2\). In words, the first reading wins for every flight time strictly less than \(2\) minutes, which in symbols is \(x < 2\). Since \(2\) itself is not included, the largest integer that fits is the biggest whole number below \(2\). A quick check confirms it. At \(x = 1\) the readings are \(6 > 3\), which is true, while at \(x = 2\) they tie at \(5 > 5\), which is false. So the largest integer \(x\) can be is \(\boxed{1}\).
Review 12. A batch of \(x\) panels uses \(3(x - 2)\) ounces of grout because the first \(2\) panels are scrapped, and the bucket holds \(18\) ounces, so \(3(x - 2) \le 18\). Distribute, then solve, dividing by the positive \(3\) with no flip. What is the largest value \(x\) can be?
Show solution
Distribute the \(3\) to get \(3x - 6 \le 18\). Add \(6\) to both sides, which gives \(3x \le 24\). Now divide both sides by \(3\). Because \(3\) is positive, there is no sign flip, so you keep \(\le\) and get \(x \le 8\). In words, x can be any value up to and including \(8\), written \(x \le 8\). The bound is closed, so \(8\) itself works. Check it, \(3(8 - 2) = 18\), and \(18 \le 18\) is true, while \(x = 9\) gives \(3(7) = 21\), which is too much. The largest value x can be is \(\boxed{8}\).
Review 13. A ferry logs every stop \(x\) that is at least \(-5\) and strictly west of \(4\), so \(-5 \le x < 4\). The left end is closed, so \(-5\) counts, and the right end is open, so \(4\) does not. How many integer positions get logged?
Show solution
The compound inequality is \(-5 \le x < 4\). No multiplying or dividing by a negative happens here, so there is no sign flip to worry about, the inequality already shows the range directly. In words, \(x\) is at least \(-5\) and also strictly less than \(4\). Because the left end is \(\le\), the value \(-5\) is included. Because the right end is \(<\), the value \(4\) is excluded, so the largest integer allowed is \(3\). The integers that satisfy it are \(-5, -4, -3, -2, -1, 0, 1, 2, 3\). Counting them gives \(9\). You can also count quickly. From \(-5\) to \(3\) the gap is \(3 - (-5) = 8\), and counting both endpoints adds one more, so \(8 + 1 = 9\). The number of integer positions logged is \(\boxed{9}\).
Review 14. A candle stands \(30\) cm tall and burns down \(2.5\) cm each hour, so after \(h\) hours its height is \(30 - 2.5h\), and it must stay at least \(5\) cm tall to sit in its holder, giving \(30 - 2.5h \ge 5\). Dividing by \(-2.5\) flips the sign. What is the most whole hours the candle can burn?
Show solution
Begin with the condition for staying in the holder, \(30 - 2.5h \ge 5\). Subtract \(30\) from both sides to isolate the \(h\) term, which gives \(-2.5h \ge -25\). Now divide both sides by \(-2.5\). Dividing by a negative number flips the inequality, so \(\ge\) turns into \(\le\), and you get \(h \le 10\). In words, the candle stays tall enough for any burning time up to and including \(10\) hours. Check the boundary. At \(h = 10\) the height is \(30 - 2.5 \cdot 10 = 5\) centimeters, exactly the minimum, so \(10\) hours still works. At \(h = 11\) the height is \(30 - 2.5 \cdot 11 = 2.5\) centimeters, which is below \(5\), so it would slip out. The most whole hours it can burn is \(\boxed{10}\).
